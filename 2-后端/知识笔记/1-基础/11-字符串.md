## 一、String 类

### 1. 什么是 String

**String** 是 Java 中用于表示字符串的类，位于 `java.lang` 包中（无需导入）。

```java
String name = "张三";
String message = "Hello World";
```

### 2. String 的特点

| 特点         | 说明                   |
| ---------- | -------------------- |
| **不可变性**   | String 对象一旦创建，内容不可更改 |
| **字符串常量池** | 字符串字面量存储在常量池中，可复用    |
| **线程安全**   | 因为不可变，所以天然线程安全       |

### 3. String 的创建方式

```java
// ========== 方式一：直接赋值（推荐）==========
String s1 = "Hello";
// 存储在字符串常量池中，可复用

// ========== 方式二：new 关键字 ==========
String s2 = new String("Hello");
// 存储在堆内存中，每次都创建新对象

// ========== 方式三：字符数组转字符串 ==========
char[] chars = {'H', 'e', 'l', 'l', 'o'};
String s3 = new String(chars);

// 部分字符
String s4 = new String(chars, 1, 3);  // "ell"（从索引1开始，取3个字符）

// ========== 方式四：字节数组转字符串 ==========
byte[] bytes = {72, 101, 108, 108, 111};
String s5 = new String(bytes);  // "Hello"

// 指定编码
String s6 = new String(bytes, StandardCharsets.UTF_8);
```

### 4. String 的不可变性（重要）

```java
String s = "Hello";
s = "World";  // 不是修改原字符串，而是创建了新字符串

// 内存变化：
// 1. "Hello" 对象仍然存在于常量池
// 2. "World" 是新创建的对象
// 3. s 只是改变了引用指向
```

**图示：**

    修改前：
    s ────→ "Hello"

    修改后：
    s ────→ "World"
             "Hello"  ← 仍然存在，等待垃圾回收（如果没有其他引用）

**为什么 String 设计为不可变？**

```java
// String 类被 final 修饰，不能被继承
// 内部的字符数组被 private final 修饰
public final class String {
    private final char[] value;  // JDK 8
    // private final byte[] value;  // JDK 9+（优化存储）
}

// 不可变的好处：
// 1. 安全性：不会被意外修改
// 2. 线程安全：可以在多线程中共享
// 3. 哈希缓存：hashCode 可以缓存，提高 HashMap 等的效率
// 4. 字符串常量池：可以复用字符串
```

### 5. 字符串常量池和 intern()

```java
String s1 = "Hello";
String s2 = "Hello";
String s3 = new String("Hello");
String s4 = s3.intern();  // 返回常量池中的引用

System.out.println(s1 == s2);  // true（都指向常量池）
System.out.println(s1 == s3);  // false（s3 在堆中）
System.out.println(s1 == s4);  // true（intern 返回常量池引用）

// intern() 方法：
// - 如果常量池中存在该字符串，返回常量池中的引用
// - 如果不存在，将该字符串放入常量池，返回常量池中的引用
```

### 6. 字符串比较

#### 6.1 == 和 equals() 的区别

```java
String s1 = "Hello";
String s2 = "Hello";
String s3 = new String("Hello");

// == 比较的是地址
System.out.println(s1 == s2);      // true（同一个常量池对象）
System.out.println(s1 == s3);      // false（不同对象）

// equals() 比较的是内容
System.out.println(s1.equals(s2)); // true
System.out.println(s1.equals(s3)); // true（内容相同）
```

#### 6.2 比较方法总结

| 方法                      | 说明             |
| ----------------------- | -------------- |
| `==`                    | 比较地址（是否是同一个对象） |
| `equals()`              | 比较内容（区分大小写）    |
| `equalsIgnoreCase()`    | 比较内容（忽略大小写）    |
| `compareTo()`           | 比较大小（字典序）      |
| `compareToIgnoreCase()` | 比较大小（忽略大小写）    |

```java
String s1 = "Hello";
String s2 = "hello";
String s3 = "Apple";

// equals 系列
System.out.println(s1.equals(s2));           // false
System.out.println(s1.equalsIgnoreCase(s2)); // true

// compareTo 系列
System.out.println(s1.compareTo(s3));        // 正数（H > A）
System.out.println(s1.compareTo("Hello"));   // 0（相等）
System.out.println(s3.compareTo(s1));        // 负数（A < H）
// 返回值：负数（小于）、0（等于）、正数（大于）
```

### 7. String 常用方法

#### 7.1 获取功能

| 方法                              | 说明               | 示例                                |
| ------------------------------- | ---------------- | --------------------------------- |
| `length()`                      | 获取字符串长度          | `"Hello".length()` → 5            |
| `charAt(int index)`             | 获取指定位置的字符        | `"Hello".charAt(0)` → 'H'         |
| `indexOf(String str)`           | 查找子串首次出现的位置      | `"Hello".indexOf("l")` → 2        |
| `lastIndexOf(String str)`       | 查找子串最后出现的位置      | `"Hello".lastIndexOf("l")` → 3    |
| `substring(int begin)`          | 截取从 begin 到末尾    | `"Hello".substring(2)` → "llo"    |
| `substring(int begin, int end)` | 截取 \[begin, end) | `"Hello".substring(1, 4)` → "ell" |

```java
String s = "Hello World";

System.out.println(s.length());           // 11
System.out.println(s.charAt(0));          // H
System.out.println(s.indexOf("o"));       // 4
System.out.println(s.lastIndexOf("o"));   // 7
System.out.println(s.substring(6));       // World
System.out.println(s.substring(0, 5));    // Hello

// 查找不存在的字符串返回 -1
System.out.println(s.indexOf("Java"));    // -1
```

#### 7.2 判断功能

| 方法                          | 说明                | 版本      |
| --------------------------- | ----------------- | ------- |
| `isEmpty()`                 | 判断是否为空字符串（长度为0）   | JDK 6+  |
| `isBlank()`                 | 判断是否为空白（空或只有空白字符） | JDK 11+ |
| `contains(String str)`      | 判断是否包含指定字符串       |         |
| `startsWith(String prefix)` | 判断是否以指定字符串开头      |         |
| `endsWith(String suffix)`   | 判断是否以指定字符串结尾      |         |
| `matches(String regex)`     | 判断是否匹配正则表达式       |         |

```java
String s = "Hello World";

System.out.println(s.isEmpty());             // false
System.out.println("".isEmpty());            // true
System.out.println(s.contains("World"));     // true
System.out.println(s.startsWith("Hello"));   // true
System.out.println(s.endsWith("World"));     // true

// JDK 11+：isBlank()
System.out.println("".isBlank());            // true
System.out.println("   ".isBlank());         // true（只有空格）
System.out.println(" \t\n ".isBlank());      // true（空白字符）

// 正则匹配
System.out.println("12345".matches("\\d+")); // true（纯数字）
System.out.println("abc".matches("[a-z]+")); // true（纯小写字母）
```

#### 7.3 转换功能

| 方法                         | 说明              | 版本      |
| -------------------------- | --------------- | ------- |
| `toUpperCase()`            | 转大写             |         |
| `toLowerCase()`            | 转小写             |         |
| `trim()`                   | 去除首尾空格          |         |
| `strip()`                  | 去除首尾空白（包括全角空格等） | JDK 11+ |
| `stripLeading()`           | 去除开头空白          | JDK 11+ |
| `stripTrailing()`          | 去除结尾空白          | JDK 11+ |
| `replace(old, new)`        | 替换字符或字符串        |         |
| `replaceAll(regex, new)`   | 正则替换所有          |         |
| `replaceFirst(regex, new)` | 正则替换第一个         |         |
| `split(String regex)`      | 按指定规则切割字符串      |         |

| `getBytes()`               | 转换为字节数组         |         |
| `repeat(int count)`        | 重复 n 次          | JDK 11+ |

```java
String s = "  Hello World  ";

// 大小写转换
System.out.println(s.toUpperCase());         // "  HELLO WORLD  "

// 去除空白
System.out.println(s.trim());                // "Hello World"

// 替换
System.out.println(s.replace("World", "Java"));  // "  Hello Java  "
System.out.println("a1b2c3".replaceAll("\\d", "*")); // "a*b*c*"（数字替换为*）
System.out.println("a1b2c3".replaceFirst("\\d", "*")); // "a*b2c3"

// 切割
String[] arr = "a,b,c".split(",");           // ["a", "b", "c"]
String[] arr2 = "a,,b,,c".split(",");        // ["a", "", "b", "", "c"]
String[] arr3 = "a  b  c".split("\\s+");     // ["a", "b", "c"]（按空白切割）

// 转换
char[] chars = "Hello".toCharArray();        // ['H', 'e', 'l', 'l', 'o']
byte[] bytes = "Hello".getBytes();           // [72, 101, 108, 108, 111]
byte[] utf8 = "你好".getBytes(StandardCharsets.UTF_8);

// JDK 11+：重复
System.out.println("Ha".repeat(3));          // "HaHaHa"
System.out.println("-".repeat(10));          // "----------"
```

## 二、StringBuilder 类

### 1. 为什么需要 StringBuilder

#### 问题：String 拼接效率低

```java
// ❌ 低效写法
String s = "";
for (int i = 0; i < 10000; i++) {
    s = s + i;  // 每次拼接都创建新对象
}
// 创建了大量临时对象，效率极低！
```

#### 原因分析

    每次拼接都会：
    1. 创建新的 String 对象
    2. 复制原字符串内容
    3. 添加新内容
    4. 原对象等待垃圾回收

    拼接 10000 次 = 创建 10000 个临时对象！

### 2. StringBuilder 是什么

**StringBuilder** 是一个**可变的字符序列**，可以在原对象上直接修改内容，不创建新对象。

| 对比       | String  | StringBuilder |
| -------- | ------- | ------------- |
| **可变性**  | 不可变     | 可变            |
| **效率**   | 拼接效率低   | 拼接效率高         |
| **线程安全** | 安全      | 不安全           |
| **使用场景** | 字符串不常变化 | 频繁拼接/修改       |

### 3. StringBuilder 的创建

```java
// ========== 方式一：空参构造 ==========
StringBuilder sb1 = new StringBuilder();
// 默认容量：16 个字符

// ========== 方式二：指定容量 ==========
StringBuilder sb2 = new StringBuilder(100);
// 初始容量：100 个字符（避免频繁扩容）

// ========== 方式三：传入字符串 ==========
StringBuilder sb3 = new StringBuilder("Hello");
// 初始内容为 "Hello"，容量为 "Hello".length() + 16 = 21
```

### 4. StringBuilder 常用方法

#### 4.1 核心方法

| 方法                                        | 说明               | 返回值          |
| ----------------------------------------- | ---------------- | ------------ |
| `append(...)`                             | 追加内容             | 返回自身（支持链式调用） |
| `insert(int offset, ...)`                 | 在指定位置插入          | 返回自身         |
| `delete(int start, int end)`              | 删除 \[start, end) | 返回自身         |
| `deleteCharAt(int index)`                 | 删除指定位置字符         | 返回自身         |
| `replace(int start, int end, String str)` | 替换指定范围           | 返回自身         |
| `reverse()`                               | 反转字符串            | 返回自身         |
| `toString()`                              | 转换为 String       | String       |

#### 4.2 代码示例

```java
StringBuilder sb = new StringBuilder();

// ========== append：追加 ==========
sb.append("Hello");
sb.append(" ");
sb.append("World");
System.out.println(sb);  // Hello World

// ========== 链式调用（推荐）==========
sb.append("!").append("!").append("!");
System.out.println(sb);  // Hello World!!!

// append 可以追加任意类型
sb.append(123).append(true).append(3.14);

// ========== insert：插入 ==========
sb.insert(0, "Say: ");
System.out.println(sb);  // Say: Hello World!!!123true3.14

// ========== delete：删除 ==========
sb.delete(0, 5);  // 删除 "Say: "
System.out.println(sb);  // Hello World!!!123true3.14

// ========== deleteCharAt：删除单个字符 ==========
sb.deleteCharAt(0);  // 删除 'H'
System.out.println(sb);  // ello World!!!123true3.14

// ========== replace：替换 ==========
StringBuilder sb2 = new StringBuilder("Hello World");
sb2.replace(6, 11, "Java");
System.out.println(sb2);  // Hello Java

// ========== reverse：反转 ==========
StringBuilder sb3 = new StringBuilder("Hello");
sb3.reverse();
System.out.println(sb3);  // olleH

// ========== toString：转为 String ==========
String s = sb3.toString();
```

#### 4.3 查询和修改方法

| 方法                              | 说明          |
| ------------------------------- | ----------- |
| `length()`                      | 获取字符长度      |
| `capacity()`                    | 获取容量        |
| `charAt(int index)`             | 获取指定位置字符    |
| `setCharAt(int index, char ch)` | 修改指定位置字符    |
| `setLength(int newLength)`      | 设置长度        |
| `ensureCapacity(int min)`       | 确保容量至少为 min |

```java
StringBuilder sb = new StringBuilder("Hello");

// 长度和容量
System.out.println(sb.length());    // 5
System.out.println(sb.capacity());  // 21（"Hello".length() + 16）

// 获取和修改字符
System.out.println(sb.charAt(0));   // H
sb.setCharAt(0, 'h');
System.out.println(sb);             // hello

// 设置长度
sb.setLength(3);
System.out.println(sb);             // hel

sb.setLength(10);
System.out.println(sb.length());    // 10（多出的位置用 \0 填充）

// 确保容量
sb.ensureCapacity(100);
System.out.println(sb.capacity());  // >= 100
```

### 5. StringBuilder vs String 效率对比

```java
// ========== String 拼接（低效）==========
long start1 = System.currentTimeMillis();
String s = "";
for (int i = 0; i < 100000; i++) {
    s = s + i;
}
long end1 = System.currentTimeMillis();
System.out.println("String 耗时：" + (end1 - start1) + "ms");
// 耗时：约 10000+ ms

// ========== StringBuilder 拼接（高效）==========
long start2 = System.currentTimeMillis();
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 100000; i++) {
    sb.append(i);
}
String result = sb.toString();
long end2 = System.currentTimeMillis();
System.out.println("StringBuilder 耗时：" + (end2 - start2) + "ms");
// 耗时：约 5-10 ms
```

**结论**：StringBuilder 比 String 拼接快 **1000 倍以上**！

## 三、StringBuffer 类

### 1. StringBuffer vs StringBuilder

| 对比       | StringBuilder | StringBuffer            |
| -------- | ------------- | ----------------------- |
| **线程安全** | ❌ 不安全         | ✅ 安全（方法加了 synchronized） |
| **效率**   | ✅ 更高          | ❌ 较低                    |
| **使用场景** | 单线程           | 多线程                     |
| **推荐程度** | ⭐⭐⭐⭐⭐         | ⭐⭐⭐                     |

```java
// StringBuffer 用法与 StringBuilder 完全相同
StringBuffer buffer = new StringBuffer();
buffer.append("Hello");
buffer.append(" World");
String s = buffer.toString();
```

### 2. 如何选择

    单线程环境 → StringBuilder（效率高）
    多线程环境 → StringBuffer（线程安全）
    不需要修改 → String（简单）

## 五、StringJoiner 类（JDK 8+）

### 1. 什么是 StringJoiner

**StringJoiner** 是 JDK 8 新增的类，用于方便地拼接字符串，支持分隔符、前缀和后缀。

```java
import java.util.StringJoiner;

// 只指定分隔符
StringJoiner sj1 = new StringJoiner(",");
sj1.add("a").add("b").add("c");
System.out.println(sj1);  // a,b,c

// 指定分隔符、前缀、后缀
StringJoiner sj2 = new StringJoiner(",", "[", "]");
sj2.add("a").add("b").add("c");
System.out.println(sj2);  // [a,b,c]

// 合并两个 StringJoiner
StringJoiner sj3 = new StringJoiner("-");
sj3.add("1").add("2");
sj1.merge(sj3);
System.out.println(sj1);  // a,b,c,1-2
```

### 2. StringJoiner vs String.join()

```java
// String.join()：简单场景
String s1 = String.join(",", "a", "b", "c");  // a,b,c

// StringJoiner：需要前缀后缀时
StringJoiner sj = new StringJoiner(",", "[", "]");
sj.add("a").add("b").add("c");
String s2 = sj.toString();  // [a,b,c]

// 集合转字符串
List<String> list = Arrays.asList("a", "b", "c");
String s3 = String.join(",", list);  // a,b,c
String s4 = list.stream().collect(Collectors.joining(",", "[", "]"));  // [a,b,c]
```

## 六、实用场景总结

```java
// 场景1：用分隔符连接数组/集合
String csv = String.join(",", "a", "b", "c");  // a,b,c

// 场景2：构建路径
String path = String.join("/", "home", "user", "documents");  // home/user/documents

// 场景3：需要前缀后缀
StringJoiner sj = new StringJoiner(", ", "SELECT ", " FROM user");
sj.add("id").add("name").add("age");
String sql = sj.toString();  // SELECT id, name, age FROM user
```

## 七、常见面试题

### 1. String 为什么不可变？

```java
// String 类被 final 修饰，不能被继承
// 内部的 char[] 数组被 private final 修饰

public final class String {
    private final char[] value;  // JDK 8
    // private final byte[] value;  // JDK 9+
}

// 好处：
// 1. 安全性：不会被意外修改
// 2. 线程安全：可以在多线程中共享
// 3. 哈希缓存：hashCode 可以缓存
// 4. 字符串常量池：可以复用字符串
```

### 2. String s = "a" + "b" + "c" 创建了几个对象？

```java
String s = "a" + "b" + "c";
// 答案：只创建 1 个对象 "abc"
// 原因：编译器会优化为 String s = "abc";
```

### 3. String s = new String("abc") 创建了几个对象？

```java
String s = new String("abc");
// 答案：1 或 2 个对象
// 1. 如果常量池中没有 "abc"，则先在常量池创建 "abc"
// 2. 然后在堆中创建 String 对象
```

### 4. 循环拼接用什么？

```java
// ❌ 错误
String s = "";
for (int i = 0; i < 1000; i++) {
    s += i;  // 每次都创建新对象
}

// ✅ 正确
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i);
}
String s = sb.toString();
```

### 5. String、StringBuilder、StringBuffer 的区别？

| 特性   | String | StringBuilder | StringBuffer |
| ---- | ------ | ------------- | ------------ |
| 可变性  | 不可变    | 可变            | 可变           |
| 线程安全 | 安全     | 不安全           | 安全           |
| 效率   | 拼接低    | 高             | 中            |
| 场景   | 少量操作   | 单线程拼接         | 多线程拼接        |
