## 一、数组介绍

### 1. 什么是数组

数组是一种**容器**，用于存储**同一类型**的多个数据。

### 2. 数组的特点

| 特点       | 说明               |
| -------- | ---------------- |
| **类型固定** | 数组只能存储同一类型的数据    |
| **长度固定** | 数组一旦创建，长度不可改变    |
| **有序存储** | 元素按索引顺序存储，索引从0开始 |
| **引用类型** | 数组是引用数据类型        |
| **连续存储** | 元素在内存中连续存放，访问效率高 |

### 3. 为什么使用数组

```java
// 不使用数组：需要定义多个变量
int score1 = 90;
int score2 = 85;
int score3 = 78;
int score4 = 92;
int score5 = 88;

// 使用数组：一个变量存储多个数据
int[] scores = {90, 85, 78, 92, 88};
```

### 4. 数组结构图解

    数组名：arr
    索引：   [0]    [1]    [2]    [3]    [4]
    元素：    10     20     30     40     50
               ↑                            ↑
            第一个元素                  最后一个元素
            索引为0                    索引为length-1

### 5. 数组 vs 集合

| 对比项  | 数组            | 集合（ArrayList） |
| ---- | ------------- | ------------- |
| 长度   | **固定**，创建后不可变 | **可变**，自动扩容   |
| 存储类型 | 基本类型和引用类型     | 只能存引用类型（包装类）  |
| 内存   | 连续内存，效率高      | 底层也是数组，但有额外开销 |
| 功能   | 功能少，需要手动实现    | 提供丰富的 API     |
| 适用场景 | 长度固定、追求性能     | 长度不定、需要频繁增删   |

```java
// 数组：长度固定
int[] arr = new int[5];
// arr.add(6);  ❌ 数组没有 add 方法

// 集合：长度可变
ArrayList<Integer> list = new ArrayList<>();
list.add(10);  // ✅ 可以添加
list.add(20);
list.remove(0);  // ✅ 可以删除
```

## 二、数组的定义和静态初始化

### 1. 静态初始化

**定义时直接指定元素的值，由系统自动计算长度**

#### 格式1：完整格式

```java
数据类型[] 数组名 = new 数据类型[]{元素1, 元素2, 元素3, ...};
```

```java
int[] arr = new int[]{10, 20, 30, 40, 50};
String[] names = new String[]{"张三", "李四", "王五"};
double[] prices = new double[]{19.9, 29.9, 39.9};
```

#### 格式2：简化格式（推荐）

```java
数据类型[] 数组名 = {元素1, 元素2, 元素3, ...};
```

```java
int[] arr = {10, 20, 30, 40, 50};
String[] names = {"张三", "李四", "王五"};
double[] prices = {19.9, 29.9, 39.9};
```

### 2. 两种定义格式

```java
// 格式1：数据类型[] 数组名（推荐）
int[] arr1;
String[] names1;

// 格式2：数据类型 数组名[]
int arr2[];
String names2[];

// 两种格式完全等价，推荐使用格式1
```

### 3. 匿名数组

**没有名字的数组，通常作为方法参数使用**

```java
// 匿名数组：直接传递给方法
printArray(new int[]{10, 20, 30});

// 注意：匿名数组必须用完整格式
// printArray({10, 20, 30});  ❌ 错误，简化格式不能用于匿名数组

public static void printArray(int[] arr) {
    for (int num : arr) {
        System.out.println(num);
    }
}
```

### 4. 静态初始化示例

```java
public class ArrayDemo {
    public static void main(String[] args) {
        // 整数数组
        int[] numbers = {1, 2, 3, 4, 5};
        
        // 浮点数数组
        double[] prices = {9.9, 19.9, 29.9};
        
        // 字符数组
        char[] chars = {'a', 'b', 'c', 'd'};
        
        // 字符串数组
        String[] names = {"张三", "李四", "王五"};
        
        // 布尔数组
        boolean[] flags = {true, false, true};
        
        // 打印数组变量（输出地址值）
        System.out.println(numbers);  // [I@1b6d3586
        System.out.println(names);    // [Ljava.lang.String;@4554617c
    }
}
```

### 5. 打印数组的含义

```java
int[] arr = {10, 20, 30};
System.out.println(arr);  // [I@1b6d3586

// 解读：
// [  → 表示是数组
// I  → 表示元素类型是int
// @  → 分隔符
// 1b6d3586 → 十六进制的内存地址

// 其他类型的标识：
// [I  → int[]
// [D  → double[]
// [C  → char[]
// [Z  → boolean[]
// [B  → byte[]
// [S  → short[]
// [J  → long[]
// [F  → float[]
// [Ljava.lang.String; → String[]
// [[I → int[][]（二维数组）
```

## 三、数组元素访问

### 1. 访问格式

```java
// 获取元素
数组名[索引]

// 修改元素
数组名[索引] = 新值;
```

### 2. 索引的特点

    索引从0开始，到length-1结束

    int[] arr = {10, 20, 30, 40, 50};

    索引：    0    1    2    3    4
    元素：   10   20   30   40   50

    第一个元素：arr[0] = 10
    最后一个元素：arr[4] = 50 或 arr[arr.length - 1]

### 3. 访问示例

```java
public class ArrayAccessDemo {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30, 40, 50};
        
        // 获取元素
        System.out.println(arr[0]);  // 10
        System.out.println(arr[2]);  // 30
        System.out.println(arr[4]);  // 50
        
        // 获取数组长度
        System.out.println(arr.length);  // 5
        
        // 获取最后一个元素
        System.out.println(arr[arr.length - 1]);  // 50
        
        // 修改元素
        arr[0] = 100;
        arr[2] = 300;
        System.out.println(arr[0]);  // 100
        System.out.println(arr[2]);  // 300
        
        // 元素参与运算
        int sum = arr[0] + arr[1];
        System.out.println(sum);  // 120
    }
}
```

### 4. 数组长度属性

```java
int[] arr = {10, 20, 30, 40, 50};

// length是数组的属性，不是方法，所以没有括号
int len = arr.length;  // 5

// 注意：length是属性，不是方法
arr.length   // ✅ 正确
arr.length() // ❌ 错误

// 对比：String 的 length 是方法
String str = "Hello";
str.length()  // ✅ 正确（String用方法）
```

## 四、数组遍历操作

### 1. 什么是遍历

遍历就是**依次访问数组中的每一个元素**

### 2. for 循环遍历

```java
int[] arr = {10, 20, 30, 40, 50};

// 正向遍历
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// 反向遍历
for (int i = arr.length - 1; i >= 0; i--) {
    System.out.println(arr[i]);
}
```

### 3. 增强 for 循环（foreach）

```java
int[] arr = {10, 20, 30, 40, 50};

// 增强for循环
for (int num : arr) {
    System.out.println(num);
}

// 语法：for (元素类型 变量名 : 数组名)
// 每次循环，变量会自动获取数组中的一个元素
```

### 4. 流式遍历（JDK 8+）

```java
import java.util.Arrays;
import java.util.stream.IntStream;

int[] arr = {10, 20, 30, 40, 50};

// 方式1：Arrays.stream()
Arrays.stream(arr).forEach(System.out::println);

// 方式2：带索引的遍历
IntStream.range(0, arr.length)
    .forEach(i -> System.out.println("索引" + i + "：" + arr[i]));

// 方式3：流式操作（过滤、映射等）
int sum = Arrays.stream(arr)
    .filter(n -> n > 20)   // 过滤：只保留大于20的
    .map(n -> n * 2)       // 映射：每个元素乘2
    .sum();                // 求和
System.out.println(sum);   // 240
```

### 5. 三种遍历方式对比

| 特性   | 普通 for | 增强 for    | 流式遍历        |
| ---- | ------ | --------- | ----------- |
| 需要索引 | ✅ 有索引i | ❌ 没有索引    | 可以有         |
| 修改元素 | ✅ 可以修改 | ❌ 不能修改原数组 | ❌ 不能修改原数组   |
| 代码简洁 | 一般     | 简洁        | 最简洁（链式调用）   |
| 适用场景 | 需要索引时  | 只需读取时     | 需要过滤/映射/聚合时 |

```java
int[] arr = {10, 20, 30};

// 普通for：可以获取索引，可以修改元素
for (int i = 0; i < arr.length; i++) {
    arr[i] = arr[i] * 2;  // 可以修改
    System.out.println("索引" + i + "：" + arr[i]);
}

// 增强for：只能读取，不能修改原数组
for (int num : arr) {
    num = num * 2;  // 这只是修改了临时变量，不影响原数组
    System.out.println(num);
}

// 流式遍历：适合复杂操作
int max = Arrays.stream(arr).max().orElse(0);
```

### 6. 遍历案例

#### 案例1：求数组元素的和

```java
int[] arr = {10, 20, 30, 40, 50};

// 方式1：普通for
int sum1 = 0;
for (int i = 0; i < arr.length; i++) {
    sum1 += arr[i];
}

// 方式2：增强for
int sum2 = 0;
for (int num : arr) {
    sum2 += num;
}

// 方式3：流式（推荐）
int sum3 = Arrays.stream(arr).sum();

System.out.println("总和：" + sum3);  // 150
```

#### 案例2：求最大值和最小值

```java
int[] arr = {15, 23, 8, 46, 12};

// 方式1：普通for
int max = arr[0];
int min = arr[0];
for (int i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
    if (arr[i] < min) min = arr[i];
}

// 方式2：流式（推荐）
int max2 = Arrays.stream(arr).max().orElse(0);
int min2 = Arrays.stream(arr).min().orElse(0);

System.out.println("最大值：" + max2);  // 46
System.out.println("最小值：" + min2);  // 8
```

#### 案例3：求平均值

```java
int[] arr = {10, 20, 30, 40, 50};

// 方式1：手动计算
int sum = 0;
for (int num : arr) {
    sum += num;
}
double avg1 = (double) sum / arr.length;

// 方式2：流式（推荐）
double avg2 = Arrays.stream(arr).average().orElse(0);

System.out.println("平均值：" + avg2);  // 30.0
```

#### 案例4：统计个数

```java
int[] arr = {15, 23, 8, 46, 12, 35, 28};

// 方式1：普通遍历
int count1 = 0;
for (int num : arr) {
    if (num > 20) count1++;
}

// 方式2：流式（推荐）
long count2 = Arrays.stream(arr).filter(n -> n > 20).count();

System.out.println("大于20的个数：" + count2);  // 4
```

## 五、数组动态初始化

### 1. 动态初始化

**只指定数组长度，由系统分配默认值**

#### 格式

```java
数据类型[] 数组名 = new 数据类型[长度];
```

#### 示例

```java
// 创建长度为5的int数组
int[] arr = new int[5];

// 创建长度为3的String数组
String[] names = new String[3];

// 创建长度为10的double数组
double[] prices = new double[10];
```

### 2. 默认值规则

| 数据类型                   | 默认值           |
| ---------------------- | ------------- |
| byte, short, int, long | 0             |
| float, double          | 0.0           |
| char                   | '\u0000'（空字符） |
| boolean                | false         |
| 引用类型（String等）          | null          |

### 3. 动态初始化示例

```java
public class DynamicArrayDemo {
    public static void main(String[] args) {
        // 整数数组，默认值为0
        int[] arr1 = new int[5];
        System.out.println(arr1[0]);  // 0
        System.out.println(arr1[4]);  // 0
        
        // 浮点数数组，默认值为0.0
        double[] arr2 = new double[3];
        System.out.println(arr2[0]);  // 0.0
        
        // 布尔数组，默认值为false
        boolean[] arr3 = new boolean[3];
        System.out.println(arr3[0]);  // false
        
        // 字符串数组，默认值为null
        String[] arr4 = new String[3];
        System.out.println(arr4[0]);  // null
        
        // 动态初始化后赋值
        arr1[0] = 10;
        arr1[1] = 20;
        arr4[0] = "张三";
        arr4[1] = "李四";
    }
}
```

### 4. 静态初始化 vs 动态初始化

| 特性   | 静态初始化                   | 动态初始化                    |
| ---- | ----------------------- | ------------------------ |
| 语法   | `int[] arr = {1, 2, 3}` | `int[] arr = new int[3]` |
| 元素值  | 直接指定                    | 系统默认值                    |
| 适用场景 | 已知元素值                   | 只知道长度                    |

### 5. 使用场景

```java
// 静态初始化：已知数据
int[] scores = {90, 85, 78, 92, 88};  // 已知学生成绩

// 动态初始化：后期赋值
Scanner sc = new Scanner(System.in);
int[] arr = new int[5];  // 先创建数组

// 通过键盘录入赋值
for (int i = 0; i < arr.length; i++) {
    System.out.print("请输入第" + (i + 1) + "个数：");
    arr[i] = sc.nextInt();
}
```

## 七、数组常见问题

### 1. 数组索引越界异常

```java
int[] arr = {10, 20, 30};

// 访问不存在的索引
System.out.println(arr[3]);  // ArrayIndexOutOfBoundsException

// 索引范围：0 ~ arr.length-1
// arr的有效索引：0, 1, 2
```

**异常信息：**

    Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: 
    Index 3 out of bounds for length 3

### 2. 空指针异常

```java
int[] arr = null;

// 操作null引用
System.out.println(arr.length);  // NullPointerException
System.out.println(arr[0]);      // NullPointerException
```

**异常信息：**

    Exception in thread "main" java.lang.NullPointerException

### 3. 如何避免异常

```java
int[] arr = {10, 20, 30};

// 避免索引越界：检查索引范围
int index = 5;
if (index >= 0 && index < arr.length) {
    System.out.println(arr[index]);
} else {
    System.out.println("索引越界");
}

// 避免空指针：检查是否为null
int[] arr2 = null;
if (arr2 != null && arr2.length > 0) {
    System.out.println(arr2.length);
} else {
    System.out.println("数组为空");
}
```

### 4. 常见错误总结

| 错误                             | 原因          | 解决方案               |
| ------------------------------ | ----------- | ------------------ |
| ArrayIndexOutOfBoundsException | 索引超出范围      | 检查索引 0 \~ length-1 |
| NullPointerException           | 数组为null     | 使用前判断是否为null       |
| 数据类型不匹配                        | 元素类型与数组类型不符 | 确保类型一致             |
| 负数索引                           | 索引为负数       | 索引必须 >= 0          |

## 八、二维数组

### 1. 什么是二维数组

二维数组是**元素为一维数组**的数组，可以理解为"表格"结构。

    一维数组：一行数据
    [10, 20, 30, 40, 50]

    二维数组：多行数据（表格）
    ┌────┬────┬────┐
    │ 10 │ 20 │ 30 │  → 第0行
    ├────┼────┼────┤
    │ 40 │ 50 │ 60 │  → 第1行
    ├────┼────┼────┤
    │ 70 │ 80 │ 90 │  → 第2行
    └────┴────┴────┘
      ↑    ↑    ↑
     第0列 第1列 第2列

### 2. 二维数组的静态初始化

```java
// 格式1：完整格式
数据类型[][] 数组名 = new 数据类型[][]{{元素}, {元素}, {元素}};

// 格式2：简化格式
数据类型[][] 数组名 = {{元素}, {元素}, {元素}};
```

**示例：**

```java
// 静态初始化
int[][] arr = {
    {10, 20, 30},
    {40, 50, 60},
    {70, 80, 90}
};

// 等价于
int[][] arr = new int[][]{{10, 20, 30}, {40, 50, 60}, {70, 80, 90}};
```

### 3. 二维数组的动态初始化


### 1. 数组转 List


## 十一、可变参数（Varargs）

### 1. 什么是可变参数

可变参数允许方法接收**任意数量**的同类型参数，底层实现就是**数组**。

### 2. 语法格式

```java
// 可变参数必须放在参数列表最后
public static void method(数据类型... 参数名) {
    // 参数名就是一个数组
}
```

### 3. 使用示例

```java
public class VarargsDemo {
    public static void main(String[] args) {
        // 传递任意数量的参数
        int sum1 = sum(1, 2, 3);
        int sum2 = sum(1, 2, 3, 4, 5);
        int sum3 = sum();  // 不传也可以
        
        System.out.println(sum1);  // 6
        System.out.println(sum2);  // 15
        System.out.println(sum3);  // 0
        
        // 也可以传递数组
        int[] arr = {10, 20, 30};
        int sum4 = sum(arr);
        System.out.println(sum4);  // 60
    }
    
    // 可变参数方法
    public static int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {  // numbers 就是数组
            total += num;
        }
        return total;
    }
}
```

### 4. 注意事项

```java
// ✅ 正确：可变参数放最后
public void method(String name, int... scores) { }

// ❌ 错误：可变参数必须放最后
// public void method(int... scores, String name) { }

// ❌ 错误：只能有一个可变参数
// public void method(int... a, double... b) { }
```

### 5. 常见应用

```java
// Arrays.asList() 就是用可变参数实现的
List<String> list = Arrays.asList("A", "B", "C");

// String.format() 也是可变参数
String result = String.format("姓名：%s，年龄：%d", "张三", 18);

// 自定义工具方法
public static void printAll(Object... args) {
    for (Object obj : args) {
        System.out.println(obj);
    }
}
```
