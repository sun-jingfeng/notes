## 一、基准测试说明

主流榜单以多个独立基准综合评估模型能力，常见指标如下：

| 基准 | 测什么 | 特点 |
| ----------------------- | ------------------------------ | ------------------------------------------ |
| **LiveBench**           | 推理、编程、数学、语言等综合   | 题库滚动更新，防止训练集污染，客观评分     |
| **MMLU / MMLU-Pro**     | 多学科知识（14 个领域）        | MMLU-Pro 难度更高，10 选 1，需多步推理     |
| **SWE-Bench Verified**  | 真实 GitHub Issue 修复能力     | 衡量实际工程编码能力的黄金标准             |
| **GPQA Diamond**        | 博士级科学问答（物理/化学/生物）| 即使专家也难以稳定答对，区分顶级模型       |
| **MATH-500 / AIME**     | 竞赛数学                       | 考察符号推理和多步计算能力                 |

> 💡 启用"思维链"（Chain-of-Thought / Extended Thinking）时，模型得分通常显著高于标准推理；排行榜上标注 *Thinking* 的分数与日常 API 调用存在差异。
