# Git 其他 - less 视图快捷键

## 基本导航

| 快捷键 | 功能 |
|--------|------|
| `空格键` | 向下翻一页 |
| `b` | 向上翻一页 |
| `↓`（下箭头键） | 向下移动一行 |
| `↑`（上箭头键） | 向上移动一行 |

---

## 快速移动

| 快捷键 | 功能 |
|--------|------|
| `d` | 向下移动半页 |
| `u` | 向上移动半页 |
| `g` | 跳到文件的开头 |
| `G` | 跳到文件的结尾 |
| `n` | 跳到下一个搜索匹配项 |
| `N` | 跳到上一个搜索匹配项 |

---

## 搜索功能

| 快捷键 | 功能 |
|--------|------|
| `/` | 后跟一个模式（可以是正则表达式），然后按回车键进行向下搜索 |
| `?` | 后跟一个模式，然后按回车键进行向上搜索 |

---

## 退出和其他操作

| 快捷键 | 功能 |
|--------|------|
| `q` | 退出 less 视图，返回到命令行 |
| `h` | 显示帮助，包括更多的命令和选项 |
