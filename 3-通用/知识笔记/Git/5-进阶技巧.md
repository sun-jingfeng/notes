# Git 进阶技巧

## 存储（Stash）

临时保存工作区和暂存区的更改，便于切换分支或处理其他任务。

### 保存

```bash
git stash                            # 保存工作区和暂存区的更改到新的存储
git stash save "<存储信息>"           # 保存并附带存储信息
git stash -u                         # 包含未追踪的文件一起存储
```

### 查看

```bash
git stash list                       # 查看存储列表
git stash show                       # 查看最近一次存储的内容摘要
git stash show -p                    # 查看最近一次存储的详细差异
git stash show stash@{<n>}           # 查看指定存储的内容摘要
```

### 应用与删除

```bash
git stash apply                      # 应用最近一次存储
git stash apply stash@{<n>}          # 应用指定存储
git stash pop                        # 应用最近一次存储，并删除存储
git stash drop                       # 删除最近一次存储
git stash drop stash@{<n>}           # 删除指定存储
git stash clear                      # 删除所有存储
git stash branch <分支名>             # 从最近一次存储创建新分支
```

## 标签（Tag）

用于标记特定的提交点，常用于版本发布。

### 查看

```bash
git tag                              # 列出所有标签
git tag -l "v1.*"                    # 按模式筛选标签
git show <标签名>                     # 查看标签详细信息
```

### 创建

```bash
git tag <标签名>                      # 在当前提交创建轻量标签
git tag <标签名> <哈希值>              # 在指定提交创建轻量标签
git tag -a <标签名> -m "<标签信息>"    # 创建附注标签（推荐）
```

### 推送

```bash
git push origin <标签名>              # 推送单个标签到远程
git push origin --tags               # 推送所有标签到远程
```

### 删除

```bash
git tag -d <标签名>                   # 删除本地标签
git push origin --delete <标签名>     # 删除远程标签
```

## 常见问题

### 撤销最近一次提交（保留修改）

```bash
git reset --soft HEAD~1
```

### 修改最近一次提交信息

```bash
git commit --amend -m "<新提交信息>"
```

### 取消追踪已追踪的文件

```bash
git rm --cached <文件名>           # 取消追踪单个文件
git rm -r --cached <目录名>        # 取消追踪整个目录
```

### 查看操作历史（reflog）

```bash
git reflog                         # 查看所有操作记录，用于恢复误操作
```

## less 视图快捷键

Git 的 `log`、`diff` 等命令默认使用 less 分页器查看内容。

### 基本导航

| 快捷键 | 功能 |
|--------|------|
| `空格键` | 向下翻一页 |
| `b` | 向上翻一页 |
| `↓` / `j` | 向下移动一行 |
| `↑` / `k` | 向上移动一行 |

### 快速移动

| 快捷键 | 功能 |
|--------|------|
| `d` | 向下移动半页 |
| `u` | 向上移动半页 |
| `g` | 跳到文件的开头 |
| `G` | 跳到文件的结尾 |
| `n` | 跳到下一个搜索匹配项 |
| `N` | 跳到上一个搜索匹配项 |

### 搜索功能

| 快捷键 | 功能 |
|--------|------|
| `/` | 后跟一个模式（可以是正则表达式），然后按回车键进行向下搜索 |
| `?` | 后跟一个模式，然后按回车键进行向上搜索 |

### 退出

| 快捷键 | 功能 |
|--------|------|
| `q` | 退出 less 视图，返回到命令行 |
| `h` | 显示帮助，包括更多的命令和选项 |
