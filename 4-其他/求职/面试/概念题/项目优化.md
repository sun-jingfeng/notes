## 整体
- **web.dev 指南**

## 速度

### 接口请求优化
- 异步请求非关键数据
- 合并关键数据请求
- 请求缓存
  - HTTP 缓存
  - 本地缓存
- 请求去重
- 数据预取

### 运行速度优化
- 避免不必要的响应式数据转换
  - 数据整体无需响应：使用普通变量（`let`/`const`）而非响应式 API
  - 数据部分无需响应：`markRaw()`、`shallowRef()`、`shallowReactive()`
- 使用 Web Workers 或生成器函数处理 JS 长任务导致的线程阻塞问题
- 预加载资源
- 中断已经不需要了的请求：`AbortController`
- 使用 CSS 或 `requestAnimationFrame` 绘制动画
- 减少排列、绘制：使用 `transform` 实现移动效果、合并 DOM 操作
- 避免强制排列
- 避免大型、复杂的布局和布局抖动
- 在 `onDeactivated` 中暂停定时器、事件监听等，在 `onActivated` 中恢复
- 按需加载、页面缓存
- 高频操作节流处理，例如滚动、移动事件
- 让传给子组件的 props 尽量保持稳定，参考 Props 稳定性
- 使用 `v-once`、`v-memo`
- 虚拟列表 / 虚拟滚动
- 使用 `computed` 缓存计算结果
- 避免在模板中使用复杂表达式
- 合理使用 `key` 属性
- `defineAsyncComponent` 异步组件

- 按需引入

- 图片压缩

- Gzip / Brotli 压缩
- 图片懒加载

- 移除未使用的 CSS（PurgeCSS）

## 代码优化

### 代码质量
- 代码格式化工具
- 逻辑复用
  - 组件封装
  - 抽离相同逻辑：
    - vue2：混入 (mixin)
    - vue3：组合式函数 (composables)
    - react：hook
- 统一控制
  - 公共样式
  - 公共函数
  - 公共常量

### 解耦
- 组件内：功能分块
- 组件级别：组件、组合式参数、提取常量
- 项目级别：微前端

### 重复操作自动化
- 批量引入
  - webpack：`require.context()`
  - vite：`import.meta.glob()`
- 动态加载组件
- 自动注册组件：`unplugin-vue-components`
- 自动导入 API：`unplugin-auto-import`
- 自动生成路由：`vite-plugin-pages` / `unplugin-vue-router`
