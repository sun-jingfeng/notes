# 数据类型与类型判断

## 一、JavaScript 的数据类型

**ECMAScript 规定了 7 种基本类型（原始类型）和 1 种引用类型。**

### 1. 基本类型（7 种）

| 类型 | 说明 | 示例 |
| --- | --- | --- |
| **string** | 字符串 | `'hello'`、`"world"` |
| **number** | 数字（含 NaN、Infinity） | `42`、`3.14`、`NaN` |
| **boolean** | 布尔值 | `true`、`false` |
| **undefined** | 未定义 | 声明未赋值、函数无返回值 |
| **null** | 空值 | 主动赋空、DOM 未找到等 |
| **symbol** | 唯一标识（ES6） | `Symbol('id')` |
| **bigint** | 大整数（ES2020） | `9007199254740991n` |

基本类型存储在**栈**中，按值访问；复制时拷贝的是值本身。

### 2. 引用类型

**object** 是唯一的引用类型，包含我们常说的：

- **Array**、**Function**、**Date**、**RegExp**
- **Map**、**Set**、**WeakMap**、**WeakSet**
- 普通对象 `{}`

引用类型在**堆**中存实际数据，变量保存的是堆地址；复制时拷贝的是引用，多引用指向同一块内存。

## 二、类型判断的四种方式

### 1. typeof

**适用**：区分基本类型（除 `null` 外）与 `function`。

| 表达式 | 结果 |
| --- | --- |
| `typeof 'a'` | `'string'` |
| `typeof 1` | `'number'` |
| `typeof true` | `'boolean'` |
| `typeof undefined` | `'undefined'` |
| `typeof null` | `'object'`（历史 bug，无法修复） |
| `typeof []` | `'object'` |
| `typeof {}` | `'object'` |
| `typeof function(){}` | `'function'` |
| `typeof Symbol()` | `'symbol'` |
| `typeof 1n` | `'bigint'` |

**注意**：`typeof` 无法区分 `null`、数组与普通对象，数组判断应用 `Array.isArray()`。

### 2. instanceof

**原理**：沿对象的原型链查找，是否存在某构造函数的 `prototype`。

```js
[] instanceof Array   // true
[] instanceof Object  // true（Array 继承自 Object）
({}) instanceof Object // true
```

**局限**：跨 iframe 时构造函数不同，同一类数组会判为 `false`；基本类型（除 `object` 包装外）无法用 `instanceof` 判断。

### 3. Object.prototype.toString.call()

**最准确的通用方式**，返回 `[object Type]` 字符串。

```js
Object.prototype.toString.call(1)        // '[object Number]'
Object.prototype.toString.call('')        // '[object String]'
Object.prototype.toString.call(null)     // '[object Null]'
Object.prototype.toString.call(undefined) // '[object Undefined]'
Object.prototype.toString.call([])        // '[object Array]'
Object.prototype.toString.call({})        // '[object Object]'
Object.prototype.toString.call(new Date()) // '[object Date]'
```

封装成通用函数时，可用正则或 `slice(8, -1)` 取出类型名。

### 4. Array.isArray()

**专门判断数组**，比 `instanceof` 更可靠（不受 iframe 影响）。

```js
Array.isArray([])           // true
Array.isArray({ length: 1 }) // false
```

## 三、面试答题要点小结

| 判断方式 | 适用场景 | 注意点 |
| --- | --- | --- |
| **typeof** | 基本类型、function | `typeof null === 'object'` |
| **instanceof** | 引用类型、继承关系 | 跨 iframe 失效 |
| **Object.prototype.toString.call()** | 所有类型精确判断 | 返回 `[object Xxx]` |
| **Array.isArray()** | 数组 | 推荐替代 `instanceof Array` |
