# ES6+ 核心特性

## 一、解构赋值

从数组或对象中提取值并赋给变量，支持默认值、重命名、嵌套。

```js
const [a, b, c = 0] = [1, 2]           // a=1, b=2, c=0
const { name: n, age = 18 } = user      // 重命名 + 默认值
const { a: { b } } = { a: { b: 1 } }   // 嵌套
```

***

## 二、模板字符串

用反引号 `` ` `` 定义字符串，支持 `${expression}` 插值和多行。

```js
const name = 'Tom'
const s = `Hello, ${name}.`
const multi = `
  第一行
  第二行
`
```

***

## 三、展开运算符与剩余参数

**展开** `...`：将可迭代对象或对象“展开”为多个元素或属性。

```js
const arr = [1, 2, 3]
const arr2 = [...arr, 4]
const copy = { ...obj }
```

**剩余参数**：函数形参中收集多余实参为数组。

```js
function fn(a, b, ...rest) {
  console.log(rest)  // [3, 4, 5]
}
fn(1, 2, 3, 4, 5)
```

***

## 四、for...of 与可迭代对象

**for...of** 遍历**可迭代对象**（有 `Symbol.iterator` 且返回迭代器）：数组、字符串、Map、Set、arguments、NodeList 等。与 `for...in`（遍历键/下标）不同，for...of 遍历**值**。

```js
for (const v of [1, 2, 3]) console.log(v)  // 1 2 3
for (const [k, v] of map) console.log(k, v)
```

***

## 五、Symbol

**Symbol** 创建**唯一**的原始值，常用作对象键，避免属性名冲突。

```js
const id = Symbol('id')
obj[id] = 1
Object.getOwnPropertySymbols(obj)  // 获取 Symbol 键
```

***

## 六、Map 与 Set

| 结构 | 特点 |
| --- | --- |
| **Map** | 键值对；键可为任意类型；保持插入顺序；`get/set/has/delete/size` |
| **Set** | 值唯一、无重复；`add/has/delete/size` |

***

## 七、Proxy 与 Reflect

**Proxy** 可代理对象，拦截属性的读取、赋值、in、delete 等操作。**Reflect** 提供与 Proxy 陷阱一一对应的反射方法，便于在代理中转发默认行为。Vue3 的响应式基于 Proxy 实现。

```js
const p = new Proxy(obj, {
  get(target, key) { return Reflect.get(target, key) },
  set(target, key, val) { return Reflect.set(target, key, val) }
})
```

***

## 八、可选链与空值合并

| 语法 | 作用 |
| --- | --- |
| **可选链** `?.` | 属性链中若某层为 null/undefined，短路返回 undefined，不报错 |
| **空值合并** `??` | 仅当左侧为 `null` 或 `undefined` 时取右侧默认值；与 `||` 不同，不把 0、'' 当空 |

```js
obj?.a?.b
value ?? defaultVal  // 0、'' 不会用 defaultVal
```

***

## 九、面试要点小结

| 特性 | 要点 |
| --- | --- |
| **解构** | 数组/对象解构，默认值、重命名、嵌套 |
| **模板字符串** | 反引号、${}、多行 |
| **展开/剩余** | ... 展开数组对象、剩余参数 |
| **for...of** | 遍历可迭代对象的值 |
| **Symbol** | 唯一标识、防冲突键 |
| **Map/Set** | 任意键、唯一值、有序 |
| **Proxy/Reflect** | 代理与反射，Vue3 响应式基础 |
| **?. 与 ??** | 安全链式访问、仅 null/undefined 取默认值 |
