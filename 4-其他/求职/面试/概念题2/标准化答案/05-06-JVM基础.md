# JVM 基础

## 一、内存区域

| 区域 | 说明 |
| --- | --- |
| **堆** | 对象实例、数组；GC 主要区域；分新生代（Eden + 2 个 Survivor）与老年代 |
| **虚拟机栈** | 线程私有；栈帧存局部变量表、操作数栈、方法出口等；方法调用对应入栈出栈 |
| **方法区（元空间）** | 类元信息、常量池、静态变量；JDK8 用元空间（本地内存）实现，不再用永久代 |
| **程序计数器** | 线程私有；当前执行的字节码行号；唯一不会 OOM 的区域 |
| **本地方法栈** | 为 Native 方法服务 |

***

## 二、GC 算法简表

| 算法 | 思路 | 特点 |
| --- | --- | --- |
| **标记-清除** | 标记存活对象，清除未标记 | 产生碎片 |
| **标记-整理** | 标记后把存活对象移到一端 | 无碎片，适合老年代 |
| **复制** | 存活对象复制到另一块内存 | 无碎片，空间折半，多用于新生代 |
| **分代收集** | 新生代多用复制，老年代多用标记-整理 | 按代选择算法 |

***

## 三、面试答题要点

- **堆**：存对象，分新生代、老年代，GC 主战场。
- **栈**：线程私有，栈帧对应方法调用。
- **方法区/元空间**：类信息、常量、静态变量；JDK8 后元空间在本地内存。
- **程序计数器**：当前字节码行号，线程私有。
- **GC**：标记-清除/整理、复制、分代收集。
