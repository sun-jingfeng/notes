# Bean 的作用域与生命周期

## 一、作用域

| 作用域 | 说明 |
| --- | --- |
| **singleton** | 默认；容器内单例，同一 Bean 只创建一次 |
| **prototype** | 每次获取都新建实例 |
| **request** | Web 环境；每个 HTTP 请求一个实例 |
| **session** | Web 环境；每个 Session 一个实例 |

***

## 二、生命周期（单例 Bean）

1. **实例化**（构造器）
2. **属性注入**（依赖注入）
3. **Aware 回调**（BeanNameAware、BeanFactoryAware 等）
4. **@PostConstruct**
5. **InitializingBean#afterPropertiesSet**
6. **自定义 init-method**
7. **使用**
8. **@PreDestroy**
9. **DisposableBean#destroy**
10. **自定义 destroy-method**

***

## 三、面试答题要点

- **作用域**：singleton（默认）、prototype、request、session。
- **生命周期**：实例化 → 注入 → Aware → PostConstruct → InitializingBean → init-method → 使用 → PreDestroy → destroy → destroy-method。
