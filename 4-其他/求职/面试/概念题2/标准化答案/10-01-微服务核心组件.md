# 微服务核心组件

## 一、Spring Cloud 常用组件

| 组件 | 职责 | 说明 |
| --- | --- | --- |
| **Nacos** | 注册中心 + 配置中心 | 服务注册发现、动态配置、多环境、健康检查 |
| **OpenFeign** | 声明式 HTTP 客户端 | 接口+注解替代 RestTemplate，集成负载均衡 |
| **Gateway** | API 网关 | 路由、鉴权、限流、跨域、统一入口 |
| **Sentinel** | 流量控制与熔断 | 限流、熔断降级、热点参数、系统保护 |
| **Seata** | 分布式事务 | AT、TCC、SAGA 等模式，保证跨服务一致性 |

***

## 二、选型说明

- **Nacos** 可替代 Eureka（注册）+ Config（配置），支持 AP/CP、临时/持久实例。
- **OpenFeign** 配合负载均衡器（如 LoadBalancer）做服务间调用。
- **Gateway** 基于 WebFlux，替代 Zuul，做统一网关。
- **Sentinel** 替代 Hystrix，做限流熔断。
- **Seata** 解决跨库、跨服务事务。

***

## 三、面试答题要点

- **Nacos**：注册+配置；OpenFeign：声明式调用；Gateway：网关；Sentinel：限流熔断；Seata：分布式事务。
