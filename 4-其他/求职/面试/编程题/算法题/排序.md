# 排序

## 一、冒泡排序

### 描述

重复遍历数组，每次比较相邻两元素，若前者大于后者则交换，使较大元素逐渐"冒泡"到末尾。时间复杂度 O(n²)，空间复杂度 O(1)。

### 答案

```typescript
function bubbleSort(arr: number[]): number[] {
  for (let i = 0; i < arr.length; i++) {
    for (let j = 0; j < arr.length - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        ;[arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]
      }
    }
  }
  return arr
}
```

## 二、快速排序

### 描述

选取基准元素（通常为第一个），将小于基准的放左边、大于的放右边，再对左右子数组递归排序。平均时间复杂度 O(n log n)，最坏 O(n²)。

### 答案

```typescript
function quickSort(nums: number[], left = 0, right = nums.length - 1) {
  if (left < right) {
    const midValueIndex = getMidValueIndex(left, right)
    quickSort(nums, left, midValueIndex - 1)
    quickSort(nums, midValueIndex + 1, right)
  }

  function getMidValueIndex(i_left: number, i_right: number): number {
    const midValue = nums[i_left]
    let start = i_left
    let end = i_right
    while (start < end) {
      while (start < end && nums[end] >= midValue) { // 这个一定要在上面
        end--
      }
      while (start < end && nums[start] <= midValue) { // 这个一定要在下面
        start++
      }
      ;[nums[start], nums[end]] = [nums[end], nums[start]]
    }
    ;[nums[i_left], nums[end]] = [nums[end], nums[i_left]]
    return end
  }

  return nums
}
```
