# 普通数组

## 一、最大子数组和

**链接**：[53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

### 描述

给定一个整数数组 `nums`，找出一个具有最大和的连续子数组，返回其最大和。

### 答案

```typescript
function maxSubArray(nums: number[]): number {
  let max = -Infinity
  let pre = 0
  nums.forEach(num => {
    const result = pre > 0 ? pre + num : num
    max = Math.max(max, result)
    pre = result
  })
  return max
}
```
