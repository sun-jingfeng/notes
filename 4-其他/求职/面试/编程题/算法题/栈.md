# 栈

## 一、有效的括号

**链接**：[20. 有效的括号](https://leetcode.cn/problems/valid-parentheses/)

### 描述

给定一个只包括 `(`、`)`、`{`、`}`、`[`、`]` 的字符串 `s`，判断字符串是否有效。有效字符串需满足：左括号必须用相同类型的右括号闭合；左括号必须以正确的顺序闭合。

### 答案

```typescript
type IBracket = '(' | ')' | '[' | ']' | '{' | '}'

function isValid(str: string): boolean {
  const stack: IBracket[] = []
  const charArr = str.split('') as IBracket[]
  charArr.forEach(char => {
    if (stack.at(-1) && isPair([stack.at(-1)!, char])) {
      stack.pop()
    } else {
      stack.push(char)
    }
  })
  return stack.length === 0
}

const isPair = (bracketArr: [IBracket, IBracket]): boolean => {
  const result =
    (bracketArr[0] === '(' && bracketArr[1] === ')') ||
    (bracketArr[0] === '[' && bracketArr[1] === ']') ||
    (bracketArr[0] === '{' && bracketArr[1] === '}')
  return result
}
```

## 二、每日温度

**链接**：[739. 每日温度](https://leetcode.cn/problems/daily-temperatures/)

### 描述

给定一个整数数组 `temperatures`，表示每天的温度，返回一个数组 `answer`，其中 `answer[i]` 表示在第 `i` 天之后，要等多少天才会有更高的温度。如果之后没有更高温度，则为 `0`。

### 答案

```typescript
function dailyTemperatures(temperatures: number[]): number[] {
  const result: number[] = new Array(temperatures.length).fill(0)
  const stack: number[] = [] // 暂未找到下个更高温度天数的索引，这些索引对应的温度值从栈底到栈顶递减
  
  for (let i = 0; i < temperatures.length; i++) {
    // 用当前温度去"解决"stack中所有比当前温度低的索引，直到栈顶索引对应的温度 >= 当前温度
    while (stack.length && temperatures[i] > temperatures[stack.at(-1) as number]) {
      result[stack.at(-1)!] = i - stack.at(-1)!
      stack.pop()
    }
    stack.push(i)
  }
  return result
}
```
