# 滑动窗口

## 一、无重复字符的最长子串

**链接**：[3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

### 1.1 描述

给定一个字符串 `s`，找出其中不含有重复字符的**最长子串**的长度。

### 1.2 答案

```typescript
const lengthOfLongestSubstring = function (s: string) {
  const charSet: Set<string> = new Set()
  let maxLength = 0
  let left = 0
  let right = 0
  
  while (left < s.length) {
    while (right < s.length && !charSet.has(s.charAt(right))) {
      charSet.add(s.charAt(right))
      right++
    }
    maxLength = Math.max(maxLength, right - left)
    charSet.delete(s.charAt(left))
    left++
  }
  
  return maxLength
}
```
