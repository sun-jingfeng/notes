# 贪心算法

## 一、买卖股票的最佳时机

**链接**：[121. 买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

### 1.1 描述

给定一个数组 `prices`，其中 `prices[i]` 表示股票第 `i` 天的价格。只能完成一笔交易（买入并卖出），求能获取的最大利润。

### 1.2 答案

```typescript
function maxProfit(prices: number[]): number {
  let maxIncome = -Infinity
  let minPrePrice = Infinity
  prices.forEach(price => {
    minPrePrice = Math.min(minPrePrice, price)
    maxIncome = Math.max(maxIncome, price - minPrePrice)
  })
  return maxIncome
}
```
