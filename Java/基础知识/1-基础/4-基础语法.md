## 一、注释（Comment）

### 1. 什么是注释

注释是对代码的说明和解释，**不会被编译和执行**。

**作用：**

*   提高代码可读性
*   方便他人理解代码
*   便于后期维护
*   可以临时屏蔽代码

### 2. 三种注释类型

#### （1）单行注释

```java
// 这是单行注释
int age = 18;  // 也可以写在代码后面
```

#### （2）多行注释

```java
/*
   这是多行注释
   可以注释多行内容
   不能嵌套使用
*/
```

#### （3）文档注释

```java
/**
 * 这是文档注释
 * 可以通过javadoc工具生成API文档
 * 
 * @author 作者名
 * @version 1.0
 */
public class Demo {
    /**
     * 方法说明
     * @param a 参数说明
     * @return 返回值说明
     */
    public int add(int a, int b) {
        return a + b;
    }
}
```

### 3. 注释对比表

| 类型   | 格式          | 特点        |
| ---- | ----------- | --------- |
| 单行注释 | `// 注释`     | 注释一行      |
| 多行注释 | `/* 注释 */`  | 注释多行，不能嵌套 |
| 文档注释 | `/** 注释 */` | 可生成API文档  |

### 4. Javadoc 常用标签

| 标签            | 说明        | 使用位置    |
| ------------- | --------- | ------- |
| `@author`     | 作者        | 类       |
| `@version`    | 版本        | 类       |
| `@since`      | 从哪个版本开始   | 类、方法    |
| `@param`      | 参数说明      | 方法      |
| `@return`     | 返回值说明     | 方法      |
| `@throws`     | 抛出异常说明    | 方法      |
| `@exception`  | 同 @throws | 方法      |
| `@see`        | 参考链接      | 类、方法    |
| `@deprecated` | 已过时       | 类、方法、字段 |
| `@link`       | 内联链接      | 任意位置    |
| `@code`       | 代码格式      | 任意位置    |

```java
/**
 * 用户服务类
 * 
 * @author zhangsan
 * @version 1.0
 * @since 1.0
 * @see UserDao
 */
public class UserService {
    
    /**
     * 根据ID查询用户
     * 
     * @param id 用户ID，不能为null
     * @return 用户对象，如果不存在返回null
     * @throws IllegalArgumentException 如果id为负数
     * @deprecated 请使用 {@link #findById(Long)} 代替
     * @see User
     */
    @Deprecated
    public User getUser(Long id) {
        if (id < 0) {
            throw new IllegalArgumentException("id不能为负数");
        }
        return null;
    }
}
```

| `/** + Enter`      | 生成文档注释            |
| `Ctrl + Q`         | 查看文档注释（Quick Doc） |

## 三、字面量（Literal）

### 1. 什么是字面量

字面量是在代码中**直接写出来的固定值**，是数据在程序中的书写格式。

### 2. 字面量分类

#### （1）整数字面量

```java
int a = 10;          // 十进制
int b = 0b1010;      // 二进制（0b/0B开头）
int c = 012;         // 八进制（0开头）
int d = 0xA;         // 十六进制（0x/0X开头）
long e = 100L;       // long类型（加L/l，推荐大写L）

// JDK 7+：数字下划线分隔符（提高可读性）
int million = 1_000_000;           // 一百万
long cardNumber = 1234_5678_9012_3456L;
int binary = 0b1010_1010_1010;
```

#### （2）浮点数字面量

```java
double a = 3.14;     // 小数默认是double
float b = 3.14F;     // float类型（必须加F/f）
double c = 5.12e2;   // 科学计数法：512.0
double d = 5.12E-2;  // 科学计数法：0.0512

// JDK 7+：数字下划线
double price = 1_234_567.89;
```

#### （3）字符字面量

```java
char a = 'A';        // 单个字符（单引号）
char b = '中';       // 中文字符
char c = '\n';       // 转义字符：换行
char d = '\t';       // 转义字符：制表符
char e = '\\';       // 转义字符：反斜杠
char f = '\'';       // 转义字符：单引号

// Unicode 表示法
char g = '\u0041';   // 'A'（十六进制 Unicode）
char h = 65;         // 'A'（ASCII 码值）
```

#### （4）字符串字面量

```java
String a = "Hello";       // 字符串（双引号）
String b = "你好";        // 中文字符串
String c = "";            // 空字符串
String d = "第一行\n第二行"; // 包含转义字符
```

#### （5）文本块（JDK 15+）

```java
// 传统写法：多行字符串
String html = "<html>\n" +
              "    <body>\n" +
              "        <h1>Hello</h1>\n" +
              "    </body>\n" +
              "</html>";

// JDK 15+ 文本块：更清晰
String htmlBlock = """
        <html>
            <body>
                <h1>Hello</h1>
            </body>
        </html>
        """;

// JSON 示例
String json = """
        {
            "name": "张三",
            "age": 18,
            "city": "北京"
        }
        """;

// SQL 示例
String sql = """
        SELECT id, name, age
        FROM users
        WHERE age > 18
        ORDER BY name
        """;
```

**文本块特点：**

*   使用三个双引号 `"""` 包围
*   自动处理换行和缩进
*   开始的 `"""` 后必须换行
*   保留相对缩进，去除公共前缀空格

#### （6）布尔字面量

```java
boolean a = true;    // 真
boolean b = false;   // 假
```

#### （7）空字面量

```java
String s = null;     // 空值（只能用于引用类型）
// int a = null;     // ❌ 错误：基本类型不能为null
```

## 四、变量（Variable）

### 1. 什么是变量

变量是内存中的一块存储空间，用于**存储可变的数据**。

### 2. 变量的定义格式

```java
数据类型 变量名 = 初始值;
```

**示例：**

```java
int age = 18;           // 整数变量
double price = 99.5;    // 小数变量
char gender = '男';     // 字符变量
String name = "张三";   // 字符串变量
boolean flag = true;    // 布尔变量
```

### 3. 变量的使用

```java
// 先声明，后赋值
int age;
age = 18;

// 声明的同时赋值（推荐）
int score = 95;

// 修改变量的值
int num = 10;
num = 20;  // 修改为20

// 变量参与运算
int a = 10;
int b = 20;
int sum = a + b;  // 30

// 同时声明多个同类型变量
int x = 1, y = 2, z = 3;
```

### 4. 成员变量 vs 局部变量

| 区别       | 成员变量                | 局部变量        |
| -------- | ------------------- | ----------- |
| **定义位置** | 类中，方法外              | 方法内、代码块内、形参 |
| **默认值**  | 有默认值                | 无默认值，必须初始化  |
| **作用域**  | 整个类                 | 所在的方法/代码块   |
| **内存位置** | 堆内存（随对象）            | 栈内存（随方法）    |
| **生命周期** | 随对象创建/销毁            | 随方法调用/结束    |
| **修饰符**  | 可用 public/private 等 | 不能用访问修饰符    |

```java
public class Demo {
    // 成员变量（实例变量）
    int age;                    // 默认值 0
    String name;                // 默认值 null
    private double salary;      // 可以用访问修饰符
    
    // 静态成员变量（类变量）
    static int count;
    
    public void method() {
        // 局部变量
        int num;                // 无默认值
        // System.out.println(num);  // ❌ 错误：未初始化
        num = 10;               // 必须先赋值
        System.out.println(num); // ✅ 正确
        
        // 形参也是局部变量
    }
    
    public void method2(int param) {  // param 是局部变量
        System.out.println(param);
    }
}
```

### 5. var 类型推断（JDK 10+）

```java
// JDK 10+ 局部变量类型推断
var age = 18;              // 推断为 int
var name = "张三";          // 推断为 String
var list = new ArrayList<String>();  // 推断为 ArrayList<String>
var map = new HashMap<String, Integer>();

// var 只能用于局部变量
// ❌ 错误用法：
// var field;              // 不能没有初始值
// var obj = null;         // 不能推断null
// var arr = {1, 2, 3};    // 不能推断数组字面量

// ✅ 数组正确用法：
var arr = new int[]{1, 2, 3};

// var 常用于简化复杂类型声明
// 传统写法
Map<String, List<Integer>> map1 = new HashMap<String, List<Integer>>();
// var 简化
var map2 = new HashMap<String, List<Integer>>();

// for 循环中使用
for (var i = 0; i < 10; i++) {
    System.out.println(i);
}

// 增强 for 循环
var list = List.of("A", "B", "C");
for (var item : list) {
    System.out.println(item);
}
```

**var 使用规则：**

| 规则         | 说明                     |
| ---------- | ---------------------- |
| 仅限局部变量     | 不能用于成员变量、方法参数、返回值      |
| 必须有初始值     | 声明时必须初始化，编译器需要推断类型     |
| 不能是 null   | `var x = null;` 无法推断类型 |
| 不能是 Lambda | 需要明确函数式接口类型            |

### 6. 变量的注意事项

```java
// ❌ 错误1：使用前未赋值（局部变量）
int age;
System.out.println(age);  // 编译错误

// ❌ 错误2：重复定义
int num = 10;
int num = 20;  // 编译错误

// ❌ 错误3：超出作用域
{
    int x = 10;
}
System.out.println(x);  // 编译错误，x不存在
```

### 7. 变量的作用域

```java
public class Demo {
    public static void main(String[] args) {
        int a = 10;  // a的作用域：整个main方法
        
        {
            int b = 20;  // b的作用域：仅在这个{}内
            System.out.println(a);  // ✅ 可以访问
            System.out.println(b);  // ✅ 可以访问
        }
        
        System.out.println(a);  // ✅ 可以访问
        System.out.println(b);  // ❌ 编译错误
    }
}
```

## 五、标识符（Identifier）

### 1. 什么是标识符

标识符是程序员自己定义的名字，用于给**类、方法、变量**等命名。

### 2. 命名规则（必须遵守）

| 规则  | 说明                   | 示例                            |
| --- | -------------------- | ----------------------------- |
| 组成  | 字母、数字、下划线`_`、美元符`$$` | `age`, `age2`, `_age`, `$age` |
| 开头  | 不能以数字开头              | ❌ `2age`                      |
| 关键字 | 不能使用关键字              | ❌ `class`, `int`              |
| 大小写 | 严格区分大小写              | `Age` ≠ `age`                 |
| 长度  | 无限制（但建议简洁有意义）        | -                             |

### 3. 命名规范（建议遵守）

| 类型      | 规范      | 示例                        |
| ------- | ------- | ------------------------- |
| **包名**  | 全小写     | `com.itheima.demo`        |
| **类名**  | 大驼峰     | `Student`, `UserService`  |
| **接口名** | 大驼峰     | `Runnable`, `Comparable`  |
| **方法名** | 小驼峰     | `getUserById`, `addUser`  |
| **变量名** | 小驼峰     | `studentName`, `maxValue` |
| **常量名** | 全大写+下划线 | `MAX_VALUE`, `PI`         |

### 4. 命名示例

```java
// 包名：全小写
package com.itheima.demo;

// 类名：大驼峰
public class StudentManager {
    
    // 常量：全大写+下划线
    public static final int MAX_AGE = 150;
    public static final String DEFAULT_NAME = "未命名";
    
    // 变量名：小驼峰
    private String studentName;
    private int currentAge;
    
    // 方法名：小驼峰
    public void addStudent() {
        int currentYear = 2025;
    }
    
    // 布尔变量/方法：通常用 is/has/can 开头
    private boolean isActive;
    private boolean hasPermission;
    
    public boolean isEmpty() {
        return false;
    }
}
```

### 5. 见名知意

```java
// ❌ 不好的命名
int a = 18;
String s = "张三";
int n = 100;

// ✅ 好的命名
int age = 18;
String name = "张三";
int maxScore = 100;

// ❌ 缩写过度
int stuNm = 10;      // 难以理解
String usrAddr;      // 不清晰

// ✅ 清晰的命名
int studentNumber = 10;
String userAddress;
```

## 六、数据类型

### 1. 数据类型分类

    数据类型
    ├─ 基本数据类型（8种）
    │   ├─ 整数型：byte, short, int, long
    │   ├─ 浮点型：float, double
    │   ├─ 字符型：char
    │   └─ 布尔型：boolean
    │
    └─ 引用数据类型
        ├─ 类（class）：String, Integer, 自定义类等
        ├─ 数组（array）
        ├─ 接口（interface）
        └─ 枚举（enum）

### 2. 基本数据类型详解

#### 整数类型

| 类型      | 字节 | 位数 | 范围                   | 默认值 |
| ------- | -- | -- | -------------------- | --- |
| `byte`  | 1  | 8  | -128 \~ 127          | 0   |
| `short` | 2  | 16 | -32768 \~ 32767      | 0   |
| `int`   | 4  | 32 | -2³¹ \~ 2³¹-1（约±21亿） | 0   |
| `long`  | 8  | 64 | -2⁶³ \~ 2⁶³-1        | 0L  |

#### 浮点类型

| 类型       | 字节 | 位数 | 范围       | 精度     | 默认值  |
| -------- | -- | -- | -------- | ------ | ---- |
| `float`  | 4  | 32 | ±3.4E38  | 6-7位   | 0.0F |
| `double` | 8  | 64 | ±1.7E308 | 15-16位 | 0.0  |

#### 字符类型

| 类型     | 字节 | 位数 | 范围         | 默认值      |
| ------ | -- | -- | ---------- | -------- |
| `char` | 2  | 16 | 0 \~ 65535 | '\u0000' |

#### 布尔类型

| 类型        | 取值           | 默认值   |
| --------- | ------------ | ----- |
| `boolean` | true / false | false |

### 3. 使用示例

```java
public class DataTypeDemo {
    public static void main(String[] args) {
        // 整数类型
        byte b = 100;
        short s = 10000;
        int i = 100000;
        long l = 10000000000L;  // 必须加L
        
        // 浮点类型
        float f = 3.14F;        // 必须加F
        double d = 3.14;        // 默认double
        
        // 字符类型
        char c = 'A';
        char unicode = '\u4e2d';  // '中'
        
        // 布尔类型
        boolean flag = true;
        
        // 引用类型
        String name = "张三";
    }
}
```

### 4. 包装类（Wrapper Class）

每个基本类型都有对应的包装类，用于将基本类型当作对象使用：

| 基本类型    | 包装类       | 示例                     |
| ------- | --------- | ---------------------- |
| byte    | Byte      | `Byte b = 10;`         |
| short   | Short     | `Short s = 100;`       |
| int     | Integer   | `Integer i = 1000;`    |
| long    | Long      | `Long l = 10000L;`     |
| float   | Float     | `Float f = 3.14F;`     |
| double  | Double    | `Double d = 3.14;`     |
| char    | Character | `Character c = 'A';`   |
| boolean | Boolean   | `Boolean flag = true;` |

#### 自动装箱与拆箱（JDK 5+）

```java
// 自动装箱：基本类型 → 包装类
Integer num = 100;       // 等价于 Integer.valueOf(100)
Double price = 99.9;     // 等价于 Double.valueOf(99.9)

// 自动拆箱：包装类 → 基本类型
int n = num;             // 等价于 num.intValue()
```

### 6. 类型转换注意事项


### 7. 浮点数精度问题

```java
// 浮点数存储采用 IEEE 754 标准，存在精度问题
System.out.println(0.1 + 0.2);           // 0.30000000000000004
System.out.println(0.1 + 0.2 == 0.3);    // false

// 解决方案1：使用 BigDecimal（推荐用于金额计算）
import java.math.BigDecimal;
BigDecimal a = new BigDecimal("0.1");
BigDecimal b = new BigDecimal("0.2");
System.out.println(a.add(b));            // 0.3

// 解决方案2：允许误差范围比较
double x = 0.1 + 0.2;
double y = 0.3;
double epsilon = 1e-10;
System.out.println(Math.abs(x - y) < epsilon);  // true

// ❌ 不要用 new BigDecimal(double)
BigDecimal bad = new BigDecimal(0.1);    // 仍有精度问题
// ✅ 使用字符串构造或 valueOf
BigDecimal good = new BigDecimal("0.1");
BigDecimal good2 = BigDecimal.valueOf(0.1);
```

## 九、常见错误总结

```java
// ❌ 变量未初始化（局部变量）
int a;
System.out.println(a);  // 错误

// ❌ 变量重复定义
int num = 10;
int num = 20;  // 错误

// ❌ long类型忘记加L
long big = 10000000000;   // 错误（超出int范围）
long big = 10000000000L;  // 正确

// ❌ float类型忘记加F
float f = 3.14;   // 错误
float f = 3.14F;  // 正确

// ❌ 用关键字做变量名
int class = 10;  // 错误

// ❌ 变量名以数字开头
int 2age = 18;   // 错误
int age2 = 18;   // 正确

// ❌ 超出作用域
{
    int x = 10;
}
System.out.println(x);  // 错误

// ❌ 类型不匹配
int a = 3.14;    // 错误
double a = 3.14; // 正确

// ❌ null 拆箱导致空指针
Integer num = null;
int n = num;     // 运行时 NullPointerException

// ❌ 浮点数精度比较
System.out.println(0.1 + 0.2 == 0.3);  // false，精度问题

// ❌ 包装类用 == 比较
Integer a = 200;
Integer b = 200;
System.out.println(a == b);       // false（超出缓存范围）
System.out.println(a.equals(b));  // true（应该用 equals）
```
