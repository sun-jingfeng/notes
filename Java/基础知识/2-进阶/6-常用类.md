## 一、Math 类

### 1. 什么是 Math 类

`Math` 类是 Java 中的**数学工具类**，提供了各种数学运算方法。

```java
// Math 类在 java.lang 包中，不需要导入
// Math 类的方法都是静态方法，直接通过类名调用
// Math 类的构造方法是私有的，不能创建对象
Math.abs(-10);  // 10
```

***

### 2. 常用常量

```java
// 圆周率 π（约 3.141592653589793）
double pi = Math.PI;

// 自然常数 e（约 2.718281828459045）
double e = Math.E;
```

***

### 3. 常用方法

#### 基本运算

```java
// 绝对值
Math.abs(-10);      // 10
Math.abs(-3.14);    // 3.14

// 最大值、最小值
Math.max(10, 20);   // 20
Math.min(10, 20);   // 10

// 幂运算
Math.pow(2, 3);     // 8.0（2的3次方）

// 平方根
Math.sqrt(16);      // 4.0

// 立方根
Math.cbrt(27);      // 3.0

// 指数（e 的 x 次方）
Math.exp(1);        // 2.718281828459045（e的1次方）

// 自然对数（以 e 为底）
Math.log(Math.E);   // 1.0

// 常用对数（以 10 为底）
Math.log10(100);    // 2.0
```

***

#### 取整运算

```java
// 向上取整（进一法）
Math.ceil(3.1);     // 4.0
Math.ceil(3.9);     // 4.0
Math.ceil(-3.1);    // -3.0

// 向下取整（去尾法）
Math.floor(3.1);    // 3.0
Math.floor(3.9);    // 3.0
Math.floor(-3.1);   // -4.0

// 四舍五入
Math.round(3.4);    // 3（返回 long）
Math.round(3.5);    // 4
Math.round(-3.5);   // -3（注意：-3.5 四舍五入是 -3）

// 取整（直接截断小数部分）
(int) 3.9;          // 3
(int) -3.9;         // -3
```

***

#### 三角函数

```java
// 正弦（参数为弧度）
Math.sin(Math.PI / 2);  // 1.0

// 余弦
Math.cos(0);            // 1.0

// 正切
Math.tan(Math.PI / 4);  // 1.0（约等于）

// 角度转弧度
double radians = Math.toRadians(90);  // 1.5707963267948966

// 弧度转角度
double degrees = Math.toDegrees(Math.PI);  // 180.0
```

***

#### 随机数

```java
// 返回 [0.0, 1.0) 的随机数
double random = Math.random();

// 生成 [0, 10) 的随机整数
int num1 = (int)(Math.random() * 10);

// 生成 [1, 10] 的随机整数
int num2 = (int)(Math.random() * 10) + 1;

// 生成 [min, max] 的随机整数（通用公式）
int min = 5, max = 10;
int num3 = (int)(Math.random() * (max - min + 1)) + min;
```

***

### 4. Math 类方法速查表

| ----------- | ----------- | --------------------- | ------- |

## 二、System 类

### 1. 什么是 System 类

`System` 类提供了与**系统相关的**方法和属性。

```java
// System 类在 java.lang 包中，不需要导入
// System 类的方法都是静态方法
// System 类不能被实例化
```

***

### 2. 常用方法

#### 标准输入输出

```java
// 标准输出
System.out.println("Hello");  // 输出并换行
System.out.print("Hello");    // 输出不换行
System.out.printf("年龄：%d", 18);  // 格式化输出

// 标准错误输出（红色）
System.err.println("Error");

// 标准输入
Scanner scanner = new Scanner(System.in);
```

***

#### 时间相关

```java
// 获取当前时间的毫秒值（从 1970-01-01 00:00:00 UTC 开始）
long currentTime = System.currentTimeMillis();
System.out.println(currentTime);  // 1704067200000

// 用于计算程序运行时间
long start = System.currentTimeMillis();
// ... 执行代码 ...
long end = System.currentTimeMillis();
System.out.println("耗时：" + (end - start) + "ms");

// 获取纳秒值（更精确，用于性能测试）
long nanoTime = System.nanoTime();
```

***

#### 数组复制

```java
// arraycopy(src, srcPos, dest, destPos, length)
// 从源数组的 srcPos 位置复制 length 个元素到目标数组的 destPos 位置

int[] src = {1, 2, 3, 4, 5};
int[] dest = new int[5];

System.arraycopy(src, 0, dest, 0, 5);
// dest = {1, 2, 3, 4, 5}

System.arraycopy(src, 1, dest, 2, 3);
// dest = {0, 0, 2, 3, 4}
```

***

#### 系统属性

```java
// 获取系统属性
String javaVersion = System.getProperty("java.version");  // Java 版本
String osName = System.getProperty("os.name");            // 操作系统名称

String userDir = System.getProperty("user.dir");          // 当前工作目录
String fileSeparator = System.getProperty("file.separator");  // 文件分隔符

// 获取所有系统属性
Properties props = System.getProperties();
props.forEach((key, value) -> System.out.println(key + " = " + value));

// 设置系统属性
System.setProperty("myKey", "myValue");
```

***

#### 环境变量

```java
// 获取环境变量
String path = System.getenv("PATH");      // PATH 环境变量
String javaHome = System.getenv("JAVA_HOME");  // JAVA_HOME 环境变量

// 获取所有环境变量
Map<String, String> env = System.getenv();
env.forEach((key, value) -> System.out.println(key + " = " + value));
```

***

#### 换行符

```java
// 获取系统换行符（跨平台）
String lineSeparator = System.lineSeparator();
// Windows: "\r\n"
// Linux/Mac: "\n"

// 使用示例
System.out.print("第一行" + System.lineSeparator() + "第二行");
```

***

#### 退出程序

```java

// 参数为 0 表示正常退出，非 0 表示异常退出
System.exit(0);  // 正常退出
System.exit(1);  // 异常退出
```

***

#### 垃圾回收

```java

```

## 三、Random 类

### 1. 什么是 Random 类

`Random` 类用于生成**伪随机数**，比 `Math.random()` 更灵活。

```java
import java.util.Random;
```

***

### 2. 常用方法

```java
// 创建 Random 对象
Random random = new Random();

// 使用种子创建（相同种子产生相同随机数序列）
Random random2 = new Random(12345);

// 生成随机整数（任意范围）
int num1 = random.nextInt();  // 任意 int

// 生成 [0, n) 范围的随机整数
int num2 = random.nextInt(10);  // [0, 10)

// 生成 [min, max] 范围的随机整数
int min = 5, max = 10;
int num3 = random.nextInt(max - min + 1) + min;  // [5, 10]

// 生成随机长整型
long num4 = random.nextLong();

// 生成随机浮点数 [0.0, 1.0)
double num5 = random.nextDouble();
float num6 = random.nextFloat();

// 生成随机布尔值
boolean flag = random.nextBoolean();

// 填充随机字节数组
byte[] bytes = new byte[10];
random.nextBytes(bytes);
```

***

### 3. Random vs Math.random()

| 对比项       | `Random`            | `Math.random()` |
| --------- | ------------------- | --------------- |
| **返回类型**  | 多种类型                | 只有 double       |
| **范围控制**  | `nextInt(n)` 直接指定范围 | 需要计算            |
| **可设置种子** | 可以                  | 不可以             |
| **线程安全**  | 不安全                 | 不安全             |
| **推荐使用**  | 需要多种类型或可重复序列        | 简单场景            |

***

### 4. ThreadLocalRandom（线程安全，JDK 7+）

```java
import java.util.concurrent.ThreadLocalRandom;

// 获取当前线程的随机数生成器
ThreadLocalRandom random = ThreadLocalRandom.current();

// 生成 [0, 10) 的随机整数
int num1 = random.nextInt(10);

// 生成 [5, 10) 的随机整数（直接指定范围）
int num2 = random.nextInt(5, 10);

// 生成 [1.0, 10.0) 的随机浮点数
double num3 = random.nextDouble(1.0, 10.0);
```

## 四、Objects 类

### 1. 什么是 Objects 类

`Objects` 是**对象工具类**，提供了操作对象的静态方法（JDK 7+）。

```java
import java.util.Objects;
```

***

### 2. 常用方法

#### 空值判断

```java
Object obj = null;

// 判断是否为 null
Objects.isNull(obj);      // true
Objects.nonNull(obj);     // false

// 要求非空，否则抛出 NullPointerException
Objects.requireNonNull(obj);  // 抛出 NullPointerException
Objects.requireNonNull(obj, "对象不能为空");  // 自定义异常信息

// JDK 9+：提供默认值
Objects.requireNonNullElse(obj, "默认值");  // 如果 obj 为 null，返回默认值
```

***

#### 对象比较

```java
String s1 = null;
String s2 = "hello";

// 安全的 equals 比较（避免 NullPointerException）
Objects.equals(s1, s2);  // false（不会抛异常）

// 等价于
// (s1 == s2) || (s1 != null && s1.equals(s2))

// 深度比较（用于数组）
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
Objects.deepEquals(arr1, arr2);  // true
```

***

#### 哈希值

```java
// 计算对象的哈希值（null 返回 0）
Objects.hashCode(obj);

// 计算多个对象的组合哈希值
Objects.hash("name", 18, true);  // 用于重写 hashCode()
```

***

#### 转换为字符串

```java
Object obj = null;

// null 安全的 toString
Objects.toString(obj);           // "null"
Objects.toString(obj, "默认值");  // "默认值"
```

***

### 3. Objects 类方法速查表

| 方法                    | 说明           | 示例                               |
| --------------------- | ------------ | -------------------------------- |
| `isNull(obj)`         | 判断是否为 null   | `Objects.isNull(obj)`            |
| `nonNull(obj)`        | 判断是否不为 null  | `Objects.nonNull(obj)`           |
| `requireNonNull(obj)` | 要求非空         | `Objects.requireNonNull(obj)`    |
| `equals(a, b)`        | 安全的 equals   | `Objects.equals(s1, s2)`         |
| `deepEquals(a, b)`    | 深度比较         | `Objects.deepEquals(arr1, arr2)` |
| `hashCode(obj)`       | 计算哈希值        | `Objects.hashCode(obj)`          |
| `hash(values...)`     | 组合哈希值        | `Objects.hash("name", 18)`       |
| `toString(obj)`       | 安全的 toString | `Objects.toString(obj)`          |

## 五、包装类

### 1. 什么是包装类

包装类是**基本数据类型对应的引用类型**，用于将基本类型包装成对象。

```java
// 为什么需要包装类？
// 1. 集合只能存储对象，不能存储基本类型
// 2. 提供了更多的方法操作数据
// 3. 可以表示 null 值
```

***

### 2. 基本类型与包装类对应关系

| 基本类型      | 包装类         | 说明     |
| --------- | ----------- | ------ |
| `byte`    | `Byte`      | 字节     |
| `short`   | `Short`     | 短整型    |
| `int`     | `Integer`   | **整型** |
| `long`    | `Long`      | 长整型    |
| `float`   | `Float`     | 单精度浮点  |
| `double`  | `Double`    | 双精度浮点  |
| `char`    | `Character` | **字符** |
| `boolean` | `Boolean`   | 布尔     |

***

### 3. 装箱与拆箱

```java
// 装箱：基本类型 → 包装类
// 手动装箱（不推荐）
Integer i1 = Integer.valueOf(10);
// 自动装箱（推荐）
Integer i2 = 10;  // 自动调用 Integer.valueOf(10)

// 拆箱：包装类 → 基本类型
// 手动拆箱（不推荐）
int n1 = i1.intValue();
// 自动拆箱（推荐）
int n2 = i2;  // 自动调用 i2.intValue()
```

***

### 4. Integer 常用方法

```java
// 1. 字符串转整数
int num1 = Integer.parseInt("123");  // 123
int num2 = Integer.parseInt("10", 2);  // 2（二进制转十进制）

// 2. 整数转字符串
String s1 = Integer.toString(123);  // "123"
String s2 = String.valueOf(123);    // "123"
String s3 = 123 + "";               // "123"

// 3. 进制转换
String binary = Integer.toBinaryString(10);   // "1010"（二进制）
String octal = Integer.toOctalString(10);     // "12"（八进制）
String hex = Integer.toHexString(10);         // "a"（十六进制）

// 4. 最大值、最小值
int max = Integer.MAX_VALUE;  // 2147483647
int min = Integer.MIN_VALUE;  // -2147483648

// 5. 比较
Integer a = 10, b = 20;
int result = Integer.compare(a, b);  // -1（小于）
int result2 = a.compareTo(b);        // -1
```

***

### 5. Character 常用方法

```java
// 判断字符类型
Character.isDigit('5');      // true（是否是数字）
Character.isLetter('A');     // true（是否是字母）
Character.isLetterOrDigit('A');  // true（是否是字母或数字）
Character.isWhitespace(' '); // true（是否是空白字符）
Character.isUpperCase('A');  // true（是否是大写字母）
Character.isLowerCase('a');  // true（是否是小写字母）

// 大小写转换
Character.toUpperCase('a');  // 'A'
Character.toLowerCase('A');  // 'a'
```

***

### 6. 包装类的缓存机制

```java
// Integer 缓存了 -128 到 127 的对象
Integer a = 127;
Integer b = 127;
System.out.println(a == b);  // true（同一个对象）

Integer c = 128;
Integer d = 128;
System.out.println(c == d);  // false（不同对象）

// 比较包装类的值，使用 equals()
System.out.println(c.equals(d));  // true
```

| 包装类                | 缓存范围         |
| ------------------ | ------------ |
| `Byte`             | -128 \~ 127  |
| `Short`            | -128 \~ 127  |
| `Integer`          | -128 \~ 127  |
| `Long`             | -128 \~ 127  |
| `Character`        | 0 \~ 127     |
| `Boolean`          | true / false |
| `Float` / `Double` | 无缓存          |

***

## 六、BigDecimal 类

### 1. 什么是 BigDecimal

`BigDecimal` 用于**高精度的小数运算**，解决浮点数精度丢失问题。

```java
// 浮点数精度问题
System.out.println(0.1 + 0.2);  // 0.30000000000000004（精度丢失）

// 使用 BigDecimal 解决
BigDecimal a = new BigDecimal("0.1");
BigDecimal b = new BigDecimal("0.2");
System.out.println(a.add(b));  // 0.3（精确）
```

***

### 2. 创建 BigDecimal

```java
import java.math.BigDecimal;

// ✅ 推荐：使用字符串创建
BigDecimal bd1 = new BigDecimal("0.1");

// ✅ 推荐：使用静态方法创建
BigDecimal bd2 = BigDecimal.valueOf(0.1);

// ❌ 不推荐：使用 double 创建（可能有精度问题）
BigDecimal bd3 = new BigDecimal(0.1);  // 0.1000000000000000055511...

// 常用常量
BigDecimal zero = BigDecimal.ZERO;  // 0
BigDecimal one = BigDecimal.ONE;    // 1
BigDecimal ten = BigDecimal.TEN;    // 10
```

***

### 3. 常用方法

#### 四则运算

```java
BigDecimal a = new BigDecimal("10");
BigDecimal b = new BigDecimal("3");

// 加法
BigDecimal sum = a.add(b);  // 13

// 减法
BigDecimal diff = a.subtract(b);  // 7

// 乘法
BigDecimal product = a.multiply(b);  // 30

// 除法（需要指定精度和舍入模式）
BigDecimal quotient = a.divide(b, 2, RoundingMode.HALF_UP);  // 3.33
```

***

#### 舍入模式

```java
import java.math.RoundingMode;

BigDecimal a = new BigDecimal("10");
BigDecimal b = new BigDecimal("3");

// 四舍五入
a.divide(b, 2, RoundingMode.HALF_UP);  // 3.33

// 向上取整
a.divide(b, 2, RoundingMode.UP);  // 3.34

// 向下取整
a.divide(b, 2, RoundingMode.DOWN);  // 3.33

// 银行家舍入（五取偶）
a.divide(b, 2, RoundingMode.HALF_EVEN);  // 3.33
```

***

#### 其他方法

```java
BigDecimal bd = new BigDecimal("3.14159");

// 设置精度
BigDecimal scaled = bd.setScale(2, RoundingMode.HALF_UP);  // 3.14

// 绝对值
BigDecimal abs = new BigDecimal("-3.14").abs();  // 3.14

// 取反
BigDecimal neg = bd.negate();  // -3.14159

// 去除末尾的 0
BigDecimal stripped = new BigDecimal("3.1400").stripTrailingZeros();  // 3.14

// 比较大小
int result = bd.compareTo(new BigDecimal("3.14"));
// 返回值：-1（小于）、0（等于）、1（大于）

// 转换为 double
double d = bd.doubleValue();

// 转换为 int（截断小数）
int i = bd.intValue();
```

***

### 4. BigDecimal 方法速查表

| 方法                                | 说明     | 示例                                     |
| --------------------------------- | ------ | -------------------------------------- |
| `add(BigDecimal)`                 | 加法     | `a.add(b)`                             |
| `subtract(BigDecimal)`            | 减法     | `a.subtract(b)`                        |
| `multiply(BigDecimal)`            | 乘法     | `a.multiply(b)`                        |
| `divide(BigDecimal, scale, mode)` | 除法     | `a.divide(b, 2, RoundingMode.HALF_UP)` |

| `negate()`                        | 取反     | `bd.negate()`                          |

| `stripTrailingZeros()`            | 去除末尾 0 | `bd.stripTrailingZeros()`              |

## 七、Arrays 类

### 1. 什么是 Arrays 类

`Arrays` 是**数组工具类**，提供了操作数组的各种方法。

```java
import java.util.Arrays;

// 所有方法都是静态方法
```

***

### 2. 常用方法

## 十三、ChronoUnit 类

### 1. 什么是 ChronoUnit

`ChronoUnit` 是一个**时间单位枚举**，用于计算时间差。

```java
import java.time.temporal.ChronoUnit;
```

***

### 2. 计算时间差

```java
LocalDateTime dt1 = LocalDateTime.of(2024, 1, 1, 0, 0, 0);
LocalDateTime dt2 = LocalDateTime.of(2024, 1, 10, 12, 30, 0);

// 计算相差的天数

// 计算相差的小时数

// 计算相差的分钟数
long minutes = ChronoUnit.MINUTES.between(dt1, dt2);  // 13710

// 计算相差的秒数
long seconds = ChronoUnit.SECONDS.between(dt1, dt2);  // 822600
```

***

### 3. 时间单位枚举

| 枚举值                  | 说明 |
| -------------------- | -- |
| `ChronoUnit.YEARS`   | 年  |
| `ChronoUnit.MONTHS`  | 月  |
| `ChronoUnit.WEEKS`   | 周  |
| `ChronoUnit.DAYS`    | 天  |
| `ChronoUnit.HOURS`   | 小时 |
| `ChronoUnit.MINUTES` | 分钟 |
| `ChronoUnit.SECONDS` | 秒  |
| `ChronoUnit.MILLIS`  | 毫秒 |
| `ChronoUnit.NANOS`   | 纳秒 |

***

### 4. 使用 Duration 和 Period

```java
import java.time.Duration;
import java.time.Period;

// Period：计算日期差（年月日）
LocalDate date1 = LocalDate.of(2024, 1, 1);
LocalDate date2 = LocalDate.of(2024, 3, 15);
Period period = Period.between(date1, date2);
System.out.println(period.getYears());   // 0
System.out.println(period.getMonths());  // 2
System.out.println(period.getDays());    // 14

// Duration：计算时间差（时分秒）
LocalDateTime dt1 = LocalDateTime.of(2024, 1, 1, 0, 0, 0);
LocalDateTime dt2 = LocalDateTime.of(2024, 1, 1, 12, 30, 45);
Duration duration = Duration.between(dt1, dt2);
System.out.println(duration.toHours());    // 12
System.out.println(duration.toMinutes());  // 750
System.out.println(duration.toSeconds());  // 45045
System.out.println(duration.toDays());     // 0
```
