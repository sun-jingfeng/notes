## 一、集合介绍

### 1. 什么是集合

**集合**是 Java 中用于**存储多个对象**的容器，可以动态地添加、删除元素。

### 2. 集合 vs 数组

| 对比       | 数组          | 集合         |
| -------- | ----------- | ---------- |
| **长度**   | 固定，创建后不能改变  | 可变，自动扩容    |
| **存储类型** | 基本类型 + 引用类型 | 只能引用类型（对象） |
| **元素类型** | 必须一致        | 可以不一致（不推荐） |
| **功能**   | 功能单一        | 提供丰富的方法    |

```java
// 数组：长度固定
int[] arr = new int[3];
arr[3] = 10;  // ❌ 越界错误

// 集合：长度可变
ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);
list.add(4);  // ✅ 自动扩容
```

### 3. 集合体系结构

                            Collection（单列集合）
                                  │
                  ┌───────────────┼───────────────┐
                  │               │               │
                List             Set            Queue
             (有序可重复)      (无序不重复)       (队列)
                  │               │
            ┌─────┼─────┐    ┌────┼────────┐
            │           │    │    │        │
        ArrayList  LinkedList │  TreeSet  LinkedHashSet
                          HashSet  (排序)    (插入顺序)
                           (无序)


                            Map（双列集合）
                                  │
                  ┌───────────────┼───────────────┐
                  │               │               │
               HashMap        TreeMap        LinkedHashMap
               (无序)          (排序)          (插入顺序)

### 4. 常用集合分类

| 接口       | 特点         | 常用实现类                | 使用场景 |
| -------- | ---------- | -------------------- | ---- |
| **List** | 有序、可重复、有索引 | ArrayList、LinkedList | 有序列表 |
| **Set**  | 无序、不重复、无索引 | HashSet、TreeSet      | 去重   |
| **Map**  | 键值对、键不重复   | HashMap、TreeMap      | 键值映射 |

## 二、创建集合

### 1. ArrayList（最常用）

```java
import java.util.ArrayList;

// ========== 创建方式 ==========

// 方式一：空参构造（推荐）
ArrayList<String> list1 = new ArrayList<>();

// 方式二：指定初始容量（元素较多时推荐）
ArrayList<String> list2 = new ArrayList<>(100);

// 方式三：传入另一个集合
ArrayList<String> list3 = new ArrayList<>(list1);
```

### 2. 泛型（Generic）

**泛型**：限制集合中元素的类型，在编译时检查类型安全。

```java
// ========== 不使用泛型（不推荐）==========
ArrayList list = new ArrayList();
list.add("Hello");
list.add(123);        // 可以添加任意类型
list.add(new Object());
// 取出时需要强制类型转换，容易出错

// ========== 使用泛型（推荐）==========
ArrayList<String> list = new ArrayList<>();  // JDK 7+ 菱形语法
list.add("Hello");
// list.add(123);  // ❌ 编译错误，只能添加 String
String s = list.get(0);  // 不需要强制转换
```

### 3. 泛型的注意事项

```java
// 1. 泛型只能是引用类型，不能是基本类型
ArrayList<int> list1 = new ArrayList<>();     // ❌ 错误
ArrayList<Integer> list2 = new ArrayList<>(); // ✅ 正确

// 2. 基本类型对应的包装类
// int     → Integer
// double  → Double
// char    → Character
// boolean → Boolean
// byte    → Byte
// short   → Short
// long    → Long
// float   → Float

// 3. 自动装箱和拆箱
ArrayList<Integer> list = new ArrayList<>();
list.add(10);        // 自动装箱：int → Integer
int num = list.get(0);  // 自动拆箱：Integer → int
```

### 4. 快速创建集合的方式

```java
import java.util.*;

// ========== Arrays.asList()（固定大小）==========
List<String> list1 = Arrays.asList("张三", "李四", "王五");

// 转为可修改的 ArrayList
ArrayList<String> list2 = new ArrayList<>(Arrays.asList("A", "B", "C"));
list2.add("D");  // ✅ 可以添加

// ========== List.of()（JDK 9+，不可变）==========
List<String> list3 = List.of("张三", "李四", "王五");
// list3.add("赵六");  // ❌ 不可变，不能修改
// list3.set(0, "钱七");  // ❌ 不可变

// 转为可修改的 ArrayList
ArrayList<String> list4 = new ArrayList<>(List.of("A", "B", "C"));

// ========== 其他快速创建 ==========
// Set
Set<String> set = Set.of("A", "B", "C");  // JDK 9+，不可变

// Map
Map<String, Integer> map = Map.of("张三", 18, "李四", 20);  // JDK 9+，不可变
```

## 三、集合常用成员方法

### 1. ArrayList 常用方法

#### 1.1 增删改查

| 方法                    | 说明             | 返回值     |
| --------------------- | -------------- | ------- |
| `add(E e)`            | 添加元素到末尾        | boolean |
| `add(int index, E e)` | 在指定位置插入元素      | void    |
| `remove(int index)`   | 删除指定位置的元素      | 被删除的元素  |
| `remove(Object o)`    | 删除指定元素（第一个匹配的） | boolean |
| `set(int index, E e)` | 修改指定位置的元素      | 原来的元素   |
| `get(int index)`      | 获取指定位置的元素      | E       |

```java
ArrayList<String> list = new ArrayList<>();

// ========== 添加 ==========
list.add("张三");           // 添加到末尾
list.add("李四");
list.add("王五");
list.add(1, "赵六");        // 在索引1位置插入
// 结果：[张三, 赵六, 李四, 王五]

// ========== 删除 ==========
String removed = list.remove(0);  // 删除索引0的元素，返回 "张三"
boolean success = list.remove("李四");  // 删除 "李四"，返回 true
// 结果：[赵六, 王五]

// ========== 修改 ==========
String old = list.set(0, "钱七");  // 修改索引0，返回原来的 "赵六"
// 结果：[钱七, 王五]

// ========== 查询 ==========
String name = list.get(0);  // 获取索引0的元素，返回 "钱七"
```

#### 1.2 批量操作

| 方法                                | 说明           |
| --------------------------------- | ------------ |
| `addAll(Collection c)`            | 添加另一个集合的所有元素 |
| `addAll(int index, Collection c)` | 在指定位置插入另一个集合 |
| `removeAll(Collection c)`         | 删除交集元素       |
| `retainAll(Collection c)`         | 只保留交集元素      |

```java
ArrayList<String> list1 = new ArrayList<>();
list1.add("A");
list1.add("B");

ArrayList<String> list2 = new ArrayList<>();
list2.add("C");
list2.add("D");

// 批量添加
list1.addAll(list2);
System.out.println(list1);  // [A, B, C, D]

// 在指定位置批量添加
list1.addAll(1, Arrays.asList("X", "Y"));
System.out.println(list1);  // [A, X, Y, B, C, D]
```

#### 1.3 其他常用方法

| 方法                          | 说明          |
| --------------------------- | ----------- |
| `size()`                    | 获取集合长度      |
| `isEmpty()`                 | 判断是否为空      |
| `contains(Object o)`        | 判断是否包含指定元素  |
| `indexOf(Object o)`         | 查找元素首次出现的位置 |
| `lastIndexOf(Object o)`     | 查找元素最后出现的位置 |
| `clear()`                   | 清空集合        |
| `toArray()`                 | 转换为数组       |
| `subList(int from, int to)` | 截取子列表（视图）   |

```java
ArrayList<String> list = new ArrayList<>();
list.add("张三");
list.add("李四");
list.add("王五");
list.add("李四");

// ========== 常用操作 ==========
int size = list.size();              // 4
boolean empty = list.isEmpty();       // false
boolean has = list.contains("张三");  // true
int first = list.indexOf("李四");     // 1
int last = list.lastIndexOf("李四");  // 3

// ========== 截取子列表 ==========
List<String> sub = list.subList(1, 3);  // [李四, 王五]
// 注意：subList 返回的是视图，修改会影响原列表

// ========== 转换为数组 ==========
Object[] arr1 = list.toArray();
String[] arr2 = list.toArray(new String[0]);  // 推荐写法

// ========== 清空 ==========
list.clear();
System.out.println(list.isEmpty());  // true
```

#### 1.4 排序方法

```java
ArrayList<Integer> nums = new ArrayList<>();
```

### 2. 集合遍历方式

#### 2.1 普通 for 循环（List 专用）

```java
ArrayList<String> list = new ArrayList<>();
list.add("张三");
list.add("李四");
list.add("王五");

for (int i = 0; i < list.size(); i++) {
    String name = list.get(i);
    System.out.println(i + ": " + name);
}
```

#### 2.2 增强 for 循环（推荐）

```java
for (String name : list) {
    System.out.println(name);
}
```

#### 2.3 迭代器

```java
import java.util.Iterator;

Iterator<String> it = list.iterator();
while (it.hasNext()) {
    String name = it.next();
    System.out.println(name);
}
```

#### 2.4 Lambda 表达式（Java 8+）

```java
// forEach + Lambda
list.forEach(name -> System.out.println(name));

// forEach + 方法引用
list.forEach(System.out::println);

// 带索引遍历（使用 IntStream）
IntStream.range(0, list.size())
    .forEach(i -> System.out.println(i + ": " + list.get(i)));
```

#### 2.5 Stream 流（Java 8+）

```java
// 基本遍历
list.stream().forEach(System.out::println);

// 过滤后遍历
list.stream()
    .filter(name -> name.startsWith("张"))
    .forEach(System.out::println);
```

### 3. 遍历时删除元素

```java
ArrayList<String> list = new ArrayList<>();
list.add("张三");
list.add("李四");
list.add("李四");
list.add("王五");

// ❌ 错误写法：增强 for 循环中不能删除
for (String name : list) {
    if ("李四".equals(name)) {
        list.remove(name);  // 抛出 ConcurrentModificationException
    }
}

// ✅ 正确写法1：使用迭代器删除
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    String name = it.next();
    if ("李四".equals(name)) {
        it.remove();  // 使用迭代器的 remove 方法
    }
}

// ✅ 正确写法2：倒序遍历删除
for (int i = list.size() - 1; i >= 0; i--) {
    if ("李四".equals(list.get(i))) {
        list.remove(i);
    }
}

// ✅ 正确写法3：removeIf（JDK 8+，推荐）
list.removeIf(name -> "李四".equals(name));
// 或者
list.removeIf("李四"::equals);
```

```java
import java.util.ArrayList;
import java.util.Scanner;

public class Test {
```

### 练习3：查找和统计

```java
public class Test {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("张三");
        list.add("李四");
        list.add("张三");
        list.add("王五");
        list.add("张三");
        
        // 查找是否存在
        String target = "李四";
        if (list.contains(target)) {
            int index = list.indexOf(target);
            System.out.println(target + " 在索引 " + index + " 位置");
        } else {
            System.out.println(target + " 不存在");
        }
        
        // 统计出现次数（方式一：循环）
        int count = 0;
        for (String name : list) {
            if ("张三".equals(name)) {
                count++;
            }
        }
        System.out.println("张三 出现了 " + count + " 次");  // 3
        
        // 统计出现次数（方式二：Collections.frequency）
        int freq = Collections.frequency(list, "张三");
        System.out.println("张三 出现了 " + freq + " 次");  // 3
        
        // 统计出现次数（方式三：Stream）
        long streamCount = list.stream().filter("张三"::equals).count();
        System.out.println("张三 出现了 " + streamCount + " 次");  // 3
    }
}
```

### 练习4：排序

```java
public class Test {
    public static void main(String[] args) {
        // ========== 数字排序 ==========
        ArrayList<Integer> nums = new ArrayList<>();
        nums.add(5);
        nums.add(2);
        nums.add(8);
        nums.add(1);
        nums.add(9);
        
        // 升序
        Collections.sort(nums);
        System.out.println("升序：" + nums);  // [1, 2, 5, 8, 9]
        
        // 降序
        nums.sort(Collections.reverseOrder());
        System.out.println("降序：" + nums);  // [9, 8, 5, 2, 1]
        
        // ========== 字符串排序 ==========
        ArrayList<String> names = new ArrayList<>();
        names.add("张三丰");
        names.add("张无忌");
        names.add("赵敏");
        names.add("小昭");
        
        // 按长度排序
        names.sort(Comparator.comparingInt(String::length));
        System.out.println("按长度：" + names);  // [赵敏, 小昭, 张三丰, 张无忌]
        
        // 按拼音排序（中文）
        names.sort(Collator.getInstance(Locale.CHINESE));
        System.out.println("按拼音：" + names);
        
        // ========== 对象排序 ==========
        ArrayList<Student> students = new ArrayList<>();
        students.add(new Student("张三", 20));
        students.add(new Student("李四", 18));
        students.add(new Student("王五", 22));
        
        // 按年龄升序
        students.sort(Comparator.comparingInt(Student::getAge));
        students.forEach(s -> System.out.println(s.getName() + ": " + s.getAge()));
        // 李四: 18
        // 张三: 20
        // 王五: 22
        
        // 按年龄降序
        students.sort(Comparator.comparingInt(Student::getAge).reversed());
        
        // 先按年龄，再按姓名
        students.sort(Comparator
            .comparingInt(Student::getAge)
            .thenComparing(Student::getName));
    }
}
```

### 练习5：删除重复元素

```java
public class Test {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("张三");
        list.add("李四");
        list.add("张三");
        list.add("王五");
        list.add("李四");
        
        // 方法1：新建集合（保持顺序）
        ArrayList<String> newList = new ArrayList<>();
        for (String name : list) {
            if (!newList.contains(name)) {
                newList.add(name);
            }
        }
        System.out.println(newList);  // [张三, 李四, 王五]
        
        // 方法2：使用 LinkedHashSet（保持顺序，更简洁）
        ArrayList<String> result = new ArrayList<>(new LinkedHashSet<>(list));
        System.out.println(result);  // [张三, 李四, 王五]
        
        // 方法3：使用 Stream（JDK 8+）
        List<String> distinct = list.stream().distinct().collect(Collectors.toList());
        System.out.println(distinct);  // [张三, 李四, 王五]
    }
}
```

### 练习6：集合作为方法参数和返回值

```java
public class Test {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(1);
        list.add(2);
        list.add(3);
        list.add(4);
        list.add(5);
        
        // 获取偶数
        ArrayList<Integer> evenList = getEven(list);
        System.out.println("偶数：" + evenList);  // [2, 4]
        
        // 获取大于指定值的元素
        ArrayList<Integer> greater = getGreaterThan(list, 3);
        System.out.println("大于3：" + greater);  // [4, 5]
        
        // 获取最大最小值
        System.out.println("最大值：" + Collections.max(list));  // 5
        System.out.println("最小值：" + Collections.min(list));  // 1
    }
    
    // 返回集合中所有偶数
    public static ArrayList<Integer> getEven(ArrayList<Integer> list) {
        ArrayList<Integer> result = new ArrayList<>();
        for (Integer num : list) {
            if (num % 2 == 0) {
                result.add(num);
            }
        }
        return result;
    }
    
    // 返回大于指定值的元素
    public static ArrayList<Integer> getGreaterThan(ArrayList<Integer> list, int threshold) {
        ArrayList<Integer> result = new ArrayList<>();
        for (Integer num : list) {
            if (num > threshold) {
                result.add(num);
            }
        }
        return result;
    }
}
```

### 遍历方式对比

| 方式      | 语法                                      | 适用场景  |
| ------- | --------------------------------------- | ----- |
| 普通 for  | `for (int i = 0; i < list.size(); i++)` | 需要索引  |
| 增强 for  | `for (E e : list)`                      | 简单遍历  |
| 迭代器     | `Iterator` + `hasNext/next`             | 遍历时删除 |
| forEach | `list.forEach(lambda)`                  | 函数式编程 |
| Stream  | `list.stream().forEach()`               | 链式操作  |

### 创建方式对比

| 方式                           | 可变性      | 版本       |
| ---------------------------- | -------- | -------- |
| `new ArrayList<>()`          | 可变       | 所有版本     |
| `Arrays.asList()`            | 不可增删，可修改 | JDK 1.2+ |
| `List.of()`                  | 完全不可变    | JDK 9+   |
| `new ArrayList<>(List.of())` | 可变       | JDK 9+   |
